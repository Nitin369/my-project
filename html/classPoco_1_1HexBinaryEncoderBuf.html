Detailed Description

This streambuf encodes all data written to it in hexBinary encoding and forwards it to a connected ostream. In hexBinary encoding, each binary octet is encoded as a character tuple,
consisting of two hexadecimal digits ([0-9a-fA-F]) representing the octet code. See also: XML Schema Part 2: Datatypes (http://www.w3.org/TR/xmlschema-2/), section 3.2.15.

Note: The characters are directly written to the ostream's streambuf, thus bypassing the ostream. The ostream's state is therefore not updated to match the buffer's state.
