Creates a Cipher object for the given Cipher name. Valid cipher names depend on the OpenSSL version the library is linked with;
see the output of 

openssl enc --help

for a list of supported block and stream ciphers.

Common examples are:

  • AES: "aes-128", "aes-256"
  • DES: "des", "des3"
  • Blowfish: "bf"
