Detailed Description

This class represents a MIME media type, consisting of a top-level type, a subtype and an optional set of parameters.

The implementation conforms with RFC 2045 and RFC 2046.

Returns true if the type and subtype match the given type and subtype. If the MIME type is a range of types it matches any media type within the range (e.g. "image/*" matches any image media type, "*&zwj;/*" matches anything). Matching is case insensitive.
