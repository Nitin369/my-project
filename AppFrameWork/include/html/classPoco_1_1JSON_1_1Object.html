Detailed Description

Represents a JSON object. Object provides a representation based on shared pointers and optimized for performance. It is possible to convert Object to DynamicStruct. Conversion requires copying and therefore has performance penalty; the benefit is in improved syntax, eg:

std::string json = "{ \"test" : { "property" : "value" } }"; Parser parser; Var result = parser.parse(json);

// use pointers to avoid copying Object::Ptr object = result.extract<Object::Ptr>(); Var test = object->get("test"); // holds { "property" : "value" } Object::Ptr subObject = test.extract<Object::Ptr>(); test = subObject->get("property"); std::string val = test.toString(); // val holds "value"

// copy/convert to Poco::DynamicStruct Poco::DynamicStruct ds = *object;

val = ds["test"]["property"]; // val holds "value"
