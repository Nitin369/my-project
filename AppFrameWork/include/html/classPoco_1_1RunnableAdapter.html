Detailed Description

template<class C>
class Poco::RunnableAdapter< C >

This adapter simplifies using ordinary methods as targets for threads. Usage: RunnableAdapter<MyClass> ra(myObject, &MyObject::doSomething)); Thread thr; thr.Start(ra);

For using a freestanding or static member function as a thread target, please see the ThreadTarget class.
