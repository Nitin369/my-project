◆ value()

template<class T >
const T& Poco::Data::Column< std::list< T > >::value ( std::size_t  row) const
inline

Returns the field value in specified row. This is the std::list specialization and std::list is not the optimal solution for cases where random access is needed. However, to allow for compatibility with other containers, this functionality is provided here. To alleviate the problem, an effort is made to start iteration from beginning or end, depending on the position requested.
