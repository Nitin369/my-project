Detailed Description

This class provides a low-level interface to a Bluetooth LE peripheral device using the Bluetooth Generic Attribute Profile (GATT).

◆ disconnect()

virtual void IoT::BtLE::GATTClient::disconnect ( )
pure virtual

Connects the GATTClient to the Bluetooth LE peripheral with the given address. The address consists of six hexadecimal byte values, separated by a colon, e.g.: "68:C9:0B:06:23:09".

If mode is GATT_CONNECT_WAIT, waits for the connection to be established (or for an error to be reported).

If wait is GATT_CONNECT_NOWAIT, returns immediately. The connection state is reported via the connected event.
