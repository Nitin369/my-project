enum RSAPaddingMode

The padding mode used for RSA public key encryption.

Enumerator
RSA_PADDING_PKCS1 
RSA_PADDING_PKCS1_OAEP 

PKCS #1 v1.5 padding. This currently is the most widely used mode.

RSA_PADDING_SSLV23 

EME-OAEP as defined in PKCS #1 v2.0 with SHA-1, MGF1 and an empty encoding parameter. This mode is recommended for all new applications.

RSA_PADDING_NONE 

PKCS #1 v1.5 padding with an SSL-specific modification that denotes that the server is SSL3 capable. Raw RSA encryption. This mode should only be used to implement cryptographically sound padding modes in the application code. Encrypting user data directly with RSA is insecure.
