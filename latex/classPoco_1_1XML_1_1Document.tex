\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1Document}{Document}} interface represents the entire H\+T\+ML or \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} document. Conceptually, it is the root of the document tree, and provides the primary access to the document\textquotesingle{}s data.

Since elements, text nodes, comments, processing instructions, etc. cannot exist outside the context of a \mbox{\hyperlink{classPoco_1_1XML_1_1Document}{Document}}, the \mbox{\hyperlink{classPoco_1_1XML_1_1Document}{Document}} interface also contains the factory methods needed to create these objects.
