\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Timestamp}{Timestamp}} stores a monotonic$\ast$ time value with (theoretical) microseconds resolution. Timestamps can be compared with each other and simple arithmetics are supported.

\mbox{[}$\ast$\mbox{]} Note that \mbox{\hyperlink{classPoco_1_1Timestamp}{Timestamp}} values are only monotonic as long as the systems\textquotesingle{}s clock is monotonic as well (and not, e.\+g. set back due to time synchronization or other reasons).

Timestamps are U\+TC (Coordinated Universal Time) based and thus independent of the timezone in effect on the system.

The internal reference time is the Unix epoch, midnight, January 1, 1970.
