\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classPoco_1_1NamedEvent}{Named\+Event}} is a global synchronization object that allows one process or thread to signal an other process or thread that a certain event has happened.

Unlike an \mbox{\hyperlink{classPoco_1_1Event}{Event}}, which itself is the unit of synchronization, a \mbox{\hyperlink{classPoco_1_1NamedEvent}{Named\+Event}} refers to a named operating system resource being the unit of synchronization. In other words, there can be multiple instances of \mbox{\hyperlink{classPoco_1_1NamedEvent}{Named\+Event}} referring to the same actual synchronization object.

Named\+Events are always autoresetting.

There should not be more than one instance of \mbox{\hyperlink{classPoco_1_1NamedEvent}{Named\+Event}} for a given name in a process. Otherwise, the instances may interfere with each other.
