\doxysubsection{Detailed Description}
This is an optional extension handler for S\+A\+X2 to provide information about D\+TD declarations in an \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} document. \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} readers are not required to support this handler, and this handler is not included in the core S\+A\+X2 distribution.

Note that data-\/related D\+TD declarations (unparsed entities and notations) are already reported through the \mbox{\hyperlink{classPoco_1_1XML_1_1DTDHandler}{D\+T\+D\+Handler}} interface. If you are using the declaration handler together with a lexical handler, all of the events will occur between the start\+D\+TD and the end\+D\+TD events. To set the \mbox{\hyperlink{classPoco_1_1XML_1_1DeclHandler}{Decl\+Handler}} for an \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} reader, use the set\+Property method with the property\+Id \char`\"{}http\+://xml.\+org/sax/properties/declaration-\/handler\char`\"{}. If the reader does not support declaration events, it will throw a S\+A\+X\+Not\+Recognized\+Exception or a S\+A\+X\+Not\+Supported\+Exception when you attempt to register the handler.
