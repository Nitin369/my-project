\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename C$>$\newline
class Poco\+::\+Nullable$<$ C $>$}

\mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}} is a simple wrapper class for value types that allows objects or native type variables to have \char`\"{}null\char`\"{} value.

The class is useful for passing parameters to functions when parameters are optional and no default values should be used or when a non-\/assigned state is needed, such as in e.\+g. fetching null values from database.

A \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}} can be default constructed. In this case, the \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}} will have a Null value and \mbox{\hyperlink{classPoco_1_1Nullable_a2164d24e6d7d57a782d2bc641a4bdbdd}{is\+Null()}} will return true. Calling \mbox{\hyperlink{classPoco_1_1Nullable_a22045b5bf3691fedab7706dcf0244e61}{value()}} (without default value) on a Null object will throw a Null\+Value\+Exception.

A \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}} can also be constructed from a value. It is possible to assign a value to a \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}}, and to reset a \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}} to contain a Null value by calling \mbox{\hyperlink{classPoco_1_1Nullable_a50cf9981c73db555887e293f6cde5c96}{clear()}}.

For use with \mbox{\hyperlink{classPoco_1_1Nullable}{Nullable}}, the value type should support default construction.
