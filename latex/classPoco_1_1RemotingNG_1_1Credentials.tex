\doxysubsection{Detailed Description}
This class contains credentials for authentication, in the form of key-\/value pairs (attributes).

The content of a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Credentials}{Credentials}} object depends upon a specific authentication mechanism and is implementation specific.

In case of username/password credentials, a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Credentials}{Credentials}} object would contain a username (A\+T\+T\+R\+\_\+\+U\+S\+E\+R\+N\+A\+ME, \char`\"{}n\char`\"{}) and a password (A\+T\+T\+R\+\_\+\+P\+A\+S\+S\+W\+O\+RD, \char`\"{}p\char`\"{}) attribute.
