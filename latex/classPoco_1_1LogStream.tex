\doxysubsection{Detailed Description}
This class implements an ostream interface to a \mbox{\hyperlink{classPoco_1_1Logger}{Logger}}.

The stream\textquotesingle{}s buffer appends all characters written to it to a string. As soon as a CR or LF (std\+::endl) is written, the string is sent to the \mbox{\hyperlink{classPoco_1_1Logger}{Logger}}, with the current priority.

Usage example\+: \mbox{\hyperlink{classPoco_1_1LogStream}{Log\+Stream}} ls(some\+Logger); ls $<$$<$ \char`\"{}\+Some informational message\char`\"{} $<$$<$ std\+::endl; ls.\+error() $<$$<$ \char`\"{}\+Some error message\char`\"{} $<$$<$ std\+::endl;
