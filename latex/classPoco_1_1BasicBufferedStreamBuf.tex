\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename ch, typename tr, typename ba = Buffer\+Allocator$<$ch$>$$>$\newline
class Poco\+::\+Basic\+Buffered\+Stream\+Buf$<$ ch, tr, ba $>$}

This is an implementation of a buffered streambuf that greatly simplifies the implementation of custom streambufs of various kinds. Derived classes only have to override the methods read\+From\+Device() or write\+To\+Device().

This streambuf only supports unidirectional streams. In other words, the \mbox{\hyperlink{classPoco_1_1BasicBufferedStreamBuf}{Basic\+Buffered\+Stream\+Buf}} can be used for the implementation of an istream or an ostream, but not for an iostream.
