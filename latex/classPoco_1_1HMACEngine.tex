\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Engine$>$\newline
class Poco\+::\+H\+M\+A\+C\+Engine$<$ Engine $>$}

This class implementes the H\+M\+AC message authentication code algorithm, as specified in R\+FC 2104. The underlying \mbox{\hyperlink{classPoco_1_1DigestEngine}{Digest\+Engine}} (\mbox{\hyperlink{classPoco_1_1MD5Engine}{M\+D5\+Engine}}, \mbox{\hyperlink{classPoco_1_1SHA1Engine}{S\+H\+A1\+Engine}}, etc.) must be given as template argument. Since the \mbox{\hyperlink{classPoco_1_1HMACEngine}{H\+M\+A\+C\+Engine}} is a \mbox{\hyperlink{classPoco_1_1DigestEngine}{Digest\+Engine}}, it can be used with the Digest\+Stream class to create a H\+M\+AC for a stream.
