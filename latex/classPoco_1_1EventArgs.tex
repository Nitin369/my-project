\doxysubsection{Detailed Description}
The purpose of the \mbox{\hyperlink{classPoco_1_1EventArgs}{Event\+Args}} class is to be used as parameter when one doesn\textquotesingle{}t want to send any data.

One can use \mbox{\hyperlink{classPoco_1_1EventArgs}{Event\+Args}} as a base class for one\textquotesingle{}s own event arguments but with the arguments being a template parameter this is not necessary.
