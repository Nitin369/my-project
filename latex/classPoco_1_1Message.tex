\doxysubsection{Detailed Description}
This class represents a log message that is sent through a chain of log channels.

A \mbox{\hyperlink{classPoco_1_1Message}{Message}} contains a priority denoting the severity of the message, a source describing its origin, a text describing its meaning, the time of its creation, and an identifier of the process and thread that created the message.

Optionally a \mbox{\hyperlink{classPoco_1_1Message}{Message}} can also contain the source file path and line number of the statement generating the message.

A \mbox{\hyperlink{classPoco_1_1Message}{Message}} can also contain any number of named parameters that contain additional information about the event that caused the message.
