\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Data_1_1RecordSet}{Record\+Set}} provides access to data returned from a query. \mbox{\hyperlink{namespacePoco_1_1Data}{Data}} access indices (row and column) are 0-\/based, as usual in C++.

Recordset provides navigation methods to iterate through the recordset, retrieval methods to extract data, and methods to get metadata (type, etc.) about columns.

To work with a \mbox{\hyperlink{classPoco_1_1Data_1_1RecordSet}{Record\+Set}}, first create a \mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}}, execute it, and create the \mbox{\hyperlink{classPoco_1_1Data_1_1RecordSet}{Record\+Set}} from the \mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}}, as follows\+: \begin{DoxyVerb}Statement select(session);
select << "SELECT * FROM Person";
select.execute();
RecordSet rs(select);
\end{DoxyVerb}


The shorter way to do the above is following\+: \begin{DoxyVerb}RecordSet rs(session, "SELECT * FROM Person"[, new SimpleRowFormatter]);
\end{DoxyVerb}


The third (optional) argument passed to the Recordset constructor is a \mbox{\hyperlink{classPoco_1_1Data_1_1RowFormatter}{Row\+Formatter}} implementation. The formatter is used in conjunction with $<$$<$ operator for recordset data formating.

The number of rows in the \mbox{\hyperlink{classPoco_1_1Data_1_1RecordSet}{Record\+Set}} can be limited by specifying a limit for the \mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}}.
