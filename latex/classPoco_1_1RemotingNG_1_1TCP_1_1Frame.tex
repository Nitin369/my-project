\doxysubsection{Detailed Description}
This class represents a network frame used by the \mbox{\hyperlink{namespacePoco_1_1RemotingNG_1_1TCP}{T\+CP}} transport connection.

A frame consists of a header, followed by payload. Header format is as follows\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TCP_1_1Frame}{Frame}} Type\+: 32 bit unsigned integer.
\item \mbox{\hyperlink{classPoco_1_1Channel}{Channel}}\+: 32 bit unsigned integer.
\item Flags\+: 16 bit unsigned integer.
\item Payload Size\+: 16 bit unsigned integer.
\end{DoxyItemize}

All integers are in network byte order. Payload sizes are 16 bit, although practically only payloads up to a size of 1012 bytes are used.
