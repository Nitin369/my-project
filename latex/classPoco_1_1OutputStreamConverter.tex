\doxysubsection{Detailed Description}
This stream converts all characters written to the underlying ostream from one character encoding into another. If a character cannot be represented in out\+Encoding, default\+Char is used instead. If a byte sequence written to the stream is not valid in in\+Encoding, default\+Char is used instead and the encoding error count is incremented.
