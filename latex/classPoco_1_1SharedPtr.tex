\subsubsection*{template$<$class C, class RC = Reference\+Counter, class RP = Release\+Policy$<$\+C$>$$>$\newline
class Poco\+::\+Shared\+Ptr$<$ C, R\+C, R\+P $>$}

\mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} is a \char`\"{}smart\char`\"{} pointer for classes implementing reference counting based garbage collection. \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} is thus similar to \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}}. Unlike the \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} template, which can only be used with classes that support reference counting, \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} can be used with any class. For this to work, a \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} manages a reference count for the object it manages.

\mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} works in the following way\+: If an \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} is assigned an ordinary pointer to an object (via the constructor or the assignment operator), it takes ownership of the object and the object\textquotesingle{}s reference count is initialized to one. If the \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} is assigned another \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}}, the object\textquotesingle{}s reference count is incremented by one. The destructor of \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} decrements the object\textquotesingle{}s reference count by one and deletes the object if the reference count reaches zero. \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} supports dereferencing with both the -\/$>$ and the $\ast$ operator. An attempt to dereference a null \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} results in a Null\+Pointer\+Exception being thrown. \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} also implements all relational operators and a cast operator in case dynamic casting of the encapsulated data types is required.
