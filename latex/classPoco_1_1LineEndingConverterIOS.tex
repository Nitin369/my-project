Sets the target line ending for the converter.

Possible values are\+:
\begin{DoxyItemize}
\item N\+E\+W\+L\+I\+N\+E\+\_\+\+D\+E\+F\+A\+U\+LT (whatever is appropriate for the current platform)
\item N\+E\+W\+L\+I\+N\+E\+\_\+\+C\+R\+LF (Windows),
\item N\+E\+W\+L\+I\+N\+E\+\_\+\+LF (Unix),
\item N\+E\+W\+L\+I\+N\+E\+\_\+\+CR (Macintosh)
\end{DoxyItemize}

In theory, any character sequence can be used as newline sequence. In practice, however, only the above three make sense.

If an empty string is given, all newline characters are removed from the stream. \mbox{\Hypertarget{classPoco_1_1LineEndingConverterIOS_a88fa8681edf827834b08e1e37bc6ae8d}\label{classPoco_1_1LineEndingConverterIOS_a88fa8681edf827834b08e1e37bc6ae8d}}
