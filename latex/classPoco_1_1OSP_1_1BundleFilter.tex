\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleFilter}{Bundle\+Filter}} is used by \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleRepository}{Bundle\+Repository}} to determine which bundles found in a repository should be loaded. For each bundle found in a repository, the \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleFilter}{Bundle\+Filter}}\textquotesingle{}s \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleFilter_a1e237457b26baf8a2a67a3dd697fe140}{accept()}} method will be called, passing the \mbox{\hyperlink{classPoco_1_1OSP_1_1Bundle}{Bundle}} as argument. Only if \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleFilter_a1e237457b26baf8a2a67a3dd697fe140}{accept()}} returns true, the bundle will be loaded.
