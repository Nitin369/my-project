\doxysubsection{Detailed Description}
\mbox{\hyperlink{structMQTTClient__willOptions}{M\+Q\+T\+T\+Client\+\_\+will\+Options}} defines the M\+Q\+TT \char`\"{}\+Last Will and Testament\char`\"{} (L\+WT) settings for the client. In the event that a client unexpectedly loses its connection to the server, the server publishes the L\+WT message to the L\+WT topic on behalf of the client. This allows other clients (subscribed to the L\+WT topic) to be made aware that the client has disconnected. To enable the L\+WT function for a specific client, a valid pointer to an \mbox{\hyperlink{structMQTTClient__willOptions}{M\+Q\+T\+T\+Client\+\_\+will\+Options}} structure is passed in the \mbox{\hyperlink{structMQTTClient__connectOptions}{M\+Q\+T\+T\+Client\+\_\+connect\+Options}} structure used in the \mbox{\hyperlink{Paho_2MQTTClient_8h_a096b8153668ce1fde395e348abe2cfdd}{M\+Q\+T\+T\+Client\+\_\+connect()}} call that connects the client to the server. The pointer to \mbox{\hyperlink{structMQTTClient__willOptions}{M\+Q\+T\+T\+Client\+\_\+will\+Options}} can be set to N\+U\+LL if the L\+WT function is not required.
