\doxysubsection{Detailed Description}
This class provides a S\+A\+X2 (Simple A\+PI for \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}}) interface to expat, the \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} parser toolkit. The following S\+A\+X2 features and properties are supported\+:
\begin{DoxyItemize}
\item \href{http://xml.org/sax/features/external-general-entities}{\texttt{ http\+://xml.\+org/sax/features/external-\/general-\/entities}}
\item \href{http://xml.org/sax/features/external-parameter-entities}{\texttt{ http\+://xml.\+org/sax/features/external-\/parameter-\/entities}}
\item \href{http://xml.org/sax/features/namespaces}{\texttt{ http\+://xml.\+org/sax/features/namespaces}}
\item \href{http://xml.org/sax/features/namespace-prefixes}{\texttt{ http\+://xml.\+org/sax/features/namespace-\/prefixes}}
\item \href{http://xml.org/sax/properties/lexical-handler}{\texttt{ http\+://xml.\+org/sax/properties/lexical-\/handler}}
\item \href{http://xml.org/sax/properties/declaration-handler}{\texttt{ http\+://xml.\+org/sax/properties/declaration-\/handler}}
\end{DoxyItemize}

The following proprietary extensions are supported\+:
\begin{DoxyItemize}
\item \href{http://www.appinf.com/features/enable-partial-reads}{\texttt{ http\+://www.\+appinf.\+com/features/enable-\/partial-\/reads}} -- see \mbox{\hyperlink{classPoco_1_1XML_1_1ParserEngine_ab1c6f59348cd810a2522f34018701cc9}{Parser\+Engine\+::set\+Enable\+Partial\+Reads()}}

Set the state of a feature.

The feature name is any fully-\/qualified \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}. It is possible for an \mbox{\hyperlink{classPoco_1_1XML_1_1XMLReader}{X\+M\+L\+Reader}} to expose a feature value but to be unable to change the current value.
