\doxysubsection{Detailed Description}
This Windows-\/only channel works with the Windows NT \mbox{\hyperlink{classPoco_1_1Event}{Event}} Log service.

To work properly, the \mbox{\hyperlink{classPoco_1_1EventLogChannel}{Event\+Log\+Channel}} class requires that either the Poco\+Foundation.\+dll or the Poco\+Msg.\+dll \mbox{\hyperlink{namespacePoco_1_1Dynamic}{Dynamic}} Link Library containing the message definition resources can be found in \$\+P\+A\+TH.
