\doxysubsection{Detailed Description}
This is a parent class for classes storing a key pair, consisting of private and public key. Storage of the private key is optional.

If a private key is available, the \mbox{\hyperlink{classPoco_1_1Crypto_1_1KeyPair}{Key\+Pair}} can be used for decrypting data (encrypted with the public key) or computing secure digital signatures.
