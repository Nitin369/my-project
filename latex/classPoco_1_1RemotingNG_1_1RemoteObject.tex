\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1RemoteObject}{Remote\+Object}} forwards method calls to the local service object.

For every service object, the \mbox{\hyperlink{namespacePoco_1_1RemotingNG}{Remoting\+NG}} code generator generates a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1RemoteObject}{Remote\+Object}} that is a subclass of the service object\textquotesingle{}s interface class.
