\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Poco\+::\+Data\+::\+Copy\+Binding$<$ T $>$}

\mbox{\hyperlink{classPoco_1_1Data_1_1Binding}{Binding}} maps a value or multiple values (see \mbox{\hyperlink{classPoco_1_1Data_1_1Binding}{Binding}} specializations for S\+TL containers as well as type handlers) to database column(s). Values to be bound can be either mapped directly (by reference) or a copy can be created, depending on the value of the copy argument. To pass a reference to a variable, it is recommended to pass it to the intermediate utility function \mbox{\hyperlink{namespacePoco_1_1Data_1_1Keywords_a9ea4cc40a6a8425fc877c64e9645ca8c}{use()}}, which will create the proper binding. In cases when a reference is passed to binding, the storage it refers to must be valid at the statement execution time. To pass a copy of a variable, constant or string literal, use utility function \mbox{\hyperlink{classPoco_1_1Data_1_1CopyBinding_a3e47255af9c305936fc4ab5943ac0ee2}{bind()}}. Variables can be passed as either copies or references (i.\+e. using either \mbox{\hyperlink{namespacePoco_1_1Data_1_1Keywords_a9ea4cc40a6a8425fc877c64e9645ca8c}{use()}} or \mbox{\hyperlink{classPoco_1_1Data_1_1CopyBinding_a3e47255af9c305936fc4ab5943ac0ee2}{bind()}}).
