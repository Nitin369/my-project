\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Net_1_1RemoteSyslogListener}{Remote\+Syslog\+Listener}} implements listening for syslog messages sent over U\+DP, according to R\+FC 5424 \char`\"{}\+The Syslog Protocol\char`\"{} and R\+FC 5426 \char`\"{}\+Transmission of syslog messages over U\+D\+P\char`\"{}.

In addition, \mbox{\hyperlink{classPoco_1_1Net_1_1RemoteSyslogListener}{Remote\+Syslog\+Listener}} also supports the \char`\"{}old\char`\"{} B\+SD syslog protocol, as described in R\+FC 3164.

The \mbox{\hyperlink{classPoco_1_1Net_1_1RemoteSyslogListener}{Remote\+Syslog\+Listener}} is a subclass of \mbox{\hyperlink{classPoco_1_1SplitterChannel}{Poco\+::\+Splitter\+Channel}}. Every received log message is sent to the channels registered with \mbox{\hyperlink{classPoco_1_1SplitterChannel_a84809fef594d920c47fb3df1411e7ce2}{add\+Channel()}} or the \char`\"{}channel\char`\"{} property.

\mbox{\hyperlink{classPoco_1_1Message}{Poco\+::\+Message}} objects created by \mbox{\hyperlink{classPoco_1_1Net_1_1RemoteSyslogListener}{Remote\+Syslog\+Listener}} will have the following named parameters\+:
\begin{DoxyItemize}
\item addr\+: IP address of the host/interface sending the message.
\item host\+: host name; only for \char`\"{}new\char`\"{} syslog messages.
\item app\+: application name; only for \char`\"{}new\char`\"{} syslog messages. 
\end{DoxyItemize}
