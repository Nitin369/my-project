\doxysubsection{Detailed Description}
A generic \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Serializer}{Serializer}} using a proprietary binary serialization format based on the \mbox{\hyperlink{classPoco_1_1BinaryWriter}{Poco\+::\+Binary\+Writer}} class.

This class can be used by all \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Transport}{Transport}} implementations using a binary serialization format that does not need to be interoperable with other network protocols.

This serializer uses a \mbox{\hyperlink{classPoco_1_1BinaryWriter}{Poco\+::\+Binary\+Writer}} for serializing data in binary form. \mbox{\hyperlink{namespacePoco_1_1Data}{Data}} is always written in the host\textquotesingle{}s native endian format. To allow the deserializer finding out the endianess used by the serializer, a byte order mark is written at the beginning of the message.

Messages created by the \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1BinarySerializer}{Binary\+Serializer}} are not self describing and contain only minimal measures for detecting deserialization failures or message version incompatibilities.
