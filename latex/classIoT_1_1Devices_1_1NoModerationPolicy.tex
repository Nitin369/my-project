\subsubsection*{template$<$typename T$>$\newline
class Io\+T\+::\+Devices\+::\+No\+Moderation\+Policy$<$ T $>$}

This event moderation policy does no moderation and fires an event whenever the event variable\textquotesingle{}s value changes (the \mbox{\hyperlink{classIoT_1_1Devices_1_1NoModerationPolicy_af7686259dd102fd49842591679d7568e}{value\+Changed()}} member function is called).

This moderation policy can be used with any event value type (numeric and string).
