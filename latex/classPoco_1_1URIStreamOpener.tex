\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1URIStreamOpener}{U\+R\+I\+Stream\+Opener}} class is used to create and open input streams for resourced identified by Uniform Resource Identifiers.

For every \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} scheme used, a \mbox{\hyperlink{classPoco_1_1URIStreamFactory}{U\+R\+I\+Stream\+Factory}} must be registered. A \mbox{\hyperlink{classPoco_1_1FileStreamFactory}{File\+Stream\+Factory}} is automatically registered for file U\+R\+Is.
