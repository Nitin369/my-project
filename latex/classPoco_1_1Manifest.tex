\subsubsection*{template$<$class B$>$\newline
class Poco\+::\+Manifest$<$ B $>$}

A \mbox{\hyperlink{classPoco_1_1Manifest}{Manifest}} maintains a list of all classes contained in a dynamically loadable class library. Internally, the information is held in a map. An iterator is provided to iterate over all the classes in a \mbox{\hyperlink{classPoco_1_1Manifest}{Manifest}}.
