\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1NamedMutex}{Named\+Mutex}} (mutual exclusion) is a global synchronization mechanism used to control access to a shared resource in a concurrent (multi process) scenario. Using the \mbox{\hyperlink{classPoco_1_1ScopedLock}{Scoped\+Lock}} class is the preferred way to automatically lock and unlock a mutex.

Unlike a \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}} or a \mbox{\hyperlink{classPoco_1_1FastMutex}{Fast\+Mutex}}, which itself is the unit of synchronization, a \mbox{\hyperlink{classPoco_1_1NamedMutex}{Named\+Mutex}} refers to a named operating system resource being the unit of synchronization. In other words, there can be multiple instances of \mbox{\hyperlink{classPoco_1_1NamedMutex}{Named\+Mutex}} referring to the same actual synchronization object.

There should not be more than one instance of \mbox{\hyperlink{classPoco_1_1NamedMutex}{Named\+Mutex}} for a given name in a process. Otherwise, the instances may interfere with each other.
