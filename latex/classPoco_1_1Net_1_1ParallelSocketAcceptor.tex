\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Service\+Handler, class SR$>$\newline
class Poco\+::\+Net\+::\+Parallel\+Socket\+Acceptor$<$ Service\+Handler, S\+R $>$}

This class implements the Acceptor part of the Acceptor-\/\+Connector design pattern. Only the difference from single-\/threaded version is documented here, For full description see \mbox{\hyperlink{classPoco_1_1Net_1_1SocketAcceptor}{Poco\+::\+Net\+::\+Socket\+Acceptor}} documentation.

This is a multi-\/threaded version of \mbox{\hyperlink{classPoco_1_1Net_1_1SocketAcceptor}{Socket\+Acceptor}}, it differs from the single-\/threaded version in number of reactors (defaulting to number of processors) that can be specified at construction time and is rotated in a round-\/robin fashion by event handler. See \mbox{\hyperlink{classPoco_1_1Net_1_1ParallelSocketAcceptor_a38f858babc9853692aa2da749ce886bf}{Parallel\+Socket\+Acceptor\+::on\+Accept}} and \mbox{\hyperlink{classPoco_1_1Net_1_1ParallelSocketAcceptor_a6dc105fc3a7187e40c5784fcf24714fc}{Parallel\+Socket\+Acceptor\+::create\+Service\+Handler}} documentation and implementation for details.
