\doxysubsection{Detailed Description}
This class represents a \mbox{\hyperlink{namespacePoco_1_1RemotingNG_1_1TCP}{T\+CP}} connection between a client and a server.

A \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TCP_1_1Connection}{Connection}} is used by multiple proxies simultaneously. It is also used to send event messages from a server to a client.

A \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TCP_1_1Connection}{Connection}} is split up into multiple channels, with each channel belonging to exactly one \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Proxy}{Proxy}} or \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1EventSubscriber}{Event\+Subscriber}}.

Messages sent between the client and the server are split up into frames, with each frame being transmitted on a specific channel.

Each connection is handled by a thread. Incoming requests or event messages will typically be handled by a separate thread obtained from a \mbox{\hyperlink{classPoco_1_1ThreadPool}{Thread\+Pool}}, but this is outside the scope of the \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TCP_1_1Connection}{Connection}} object.
