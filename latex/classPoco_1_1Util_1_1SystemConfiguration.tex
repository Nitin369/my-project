\doxysubsection{Detailed Description}
This class implements a Configuration interface to various system properties and environment variables.

The following properties are supported\+:
\begin{DoxyItemize}
\item system.\+os\+Name\+: the operating system name
\item system.\+os\+Version\+: the operating system version
\item system.\+os\+Architecture\+: the operating system architecture
\item system.\+node\+Name\+: the node (or host) name
\item system.\+node\+Id\+: system ID, based on the Ethernet address (format \char`\"{}xxxxxxxxxxxx\char`\"{}) of the first Ethernet adapter found on the system.
\item system.\+current\+Dir\+: the current working directory
\item system.\+home\+Dir\+: the user\textquotesingle{}s home directory
\item system.\+temp\+Dir\+: the system\textquotesingle{}s temporary directory
\item system.\+date\+Time\+: the current U\+TC date and time, formatted in I\+SO 8601 format.
\item system.\+pid\+: the current process ID.
\item system.\+env.$<$\+N\+A\+M\+E$>$\+: the environment variable with the given $<$\+N\+A\+M\+E$>$.
\end{DoxyItemize}

An attempt to set a system variable will result in an Invalid\+Access\+Exception being thrown.

Enumerating environment variables is not supported. An attempt to call keys(\char`\"{}system.\+env\char`\"{}) will return an empty range.

Removing key is not supported. An attempt to remove a key results in a Not\+Implemented\+Exception being thrown.
