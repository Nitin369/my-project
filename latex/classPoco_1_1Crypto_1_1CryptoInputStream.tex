\doxysubsection{Detailed Description}
This stream transforms all data passing through it using the given \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoTransform}{Crypto\+Transform}}.

Use a \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoTransform}{Crypto\+Transform}} object provided by Cipher\+::create\+Encrytor() or \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher_a5ff6f0b7c3eb92c908150f51e9a9fdc7}{Cipher\+::create\+Decryptor()}} to create an encrypting or decrypting stream, respectively.
