\doxysubsection{Detailed Description}
This interface represents the basic operations for cryptographic transformations to be used with a \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoInputStream}{Crypto\+Input\+Stream}} or a \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoOutputStream}{Crypto\+Output\+Stream}}.

Implementations of this class are returned by the \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher}{Cipher}} class to perform encryption or decryption of data.
