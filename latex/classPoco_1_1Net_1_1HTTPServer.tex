\doxysubsection{Detailed Description}
A subclass of \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServer}{T\+C\+P\+Server}} that implements a full-\/featured multithreaded H\+T\+TP server.

A \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandlerFactory}{H\+T\+T\+P\+Request\+Handler\+Factory}} must be supplied. The \mbox{\hyperlink{classPoco_1_1Net_1_1ServerSocket}{Server\+Socket}} must be bound and in listening state.

To configure various aspects of the server, a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServerParams}{H\+T\+T\+P\+Server\+Params}} object can be passed to the constructor.

The server supports\+:
\begin{DoxyItemize}
\item H\+T\+T\+P/1.\+0 and H\+T\+T\+P/1.\+1
\item automatic handling of persistent connections.
\item automatic decoding/encoding of request/response message bodies using chunked transfer encoding.
\end{DoxyItemize}

Please see the \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServer}{T\+C\+P\+Server}} class for information about connection and thread handling.

See R\+FC 2616 \href{http://www.faqs.org/rfcs/rfc2616.html}{\texttt{ http\+://www.\+faqs.\+org/rfcs/rfc2616.\+html}} for more information about the H\+T\+TP protocol.
