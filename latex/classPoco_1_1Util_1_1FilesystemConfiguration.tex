\doxysubsection{Detailed Description}
An implementation of \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration}{Abstract\+Configuration}} that stores configuration data in a directory hierarchy in the filesystem.

Every period-\/separated part of a property name is represented as a directory in the filesystem, relative to the base directory. Values are stored in files named \char`\"{}data\char`\"{}.

All changes to properties are immediately persisted in the filesystem.

For example, a configuration consisting of the properties

logging.\+loggers.\+root.\+channel.\+class = \mbox{\hyperlink{classPoco_1_1ConsoleChannel}{Console\+Channel}} logging.\+loggers.\+app.\+name = \mbox{\hyperlink{classPoco_1_1Util_1_1Application}{Application}} logging.\+loggers.\+app.\+channel = c1 logging.\+formatters.\+f1.\+class = \mbox{\hyperlink{classPoco_1_1PatternFormatter}{Pattern\+Formatter}} logging.\+formatters.\+f1.\+pattern = \mbox{[}p\mbox{]} t

is stored in the filesystem as follows\+:

logging/ loggers/ root/ channel/ class/ data (\char`\"{}\+Console\+Channel\char`\"{}) app/ name/ data (\char`\"{}\+Application\char`\"{}) channel/ data (\char`\"{}c1\char`\"{}) formatters/ f1/ class/ data (\char`\"{}\+Pattern\+Formatter\char`\"{}) pattern/ data (\char`\"{}\mbox{[}\%p\mbox{]} \%t\char`\"{}) ~\newline
