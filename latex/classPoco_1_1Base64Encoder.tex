\doxysubsection{Detailed Description}
This ostream base64-\/encodes all data written to it and forwards it to a connected ostream. Always call \mbox{\hyperlink{classPoco_1_1Base64EncoderIOS_a108f0e195ad60cd12181de82b5a3cf30}{close()}} when done writing data, to ensure proper completion of the encoding operation.

Note\+: The characters are directly written to the ostream\textquotesingle{}s streambuf, thus bypassing the ostream. The ostream\textquotesingle{}s state is therefore not updated to match the buffer\textquotesingle{}s state.
