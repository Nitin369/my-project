\doxysubsection{Detailed Description}
\subsubsection*{template$<$class C, class N$>$\newline
class Poco\+::\+N\+Observer$<$ C, N $>$}

This template class implements an adapter that sits between a \mbox{\hyperlink{classPoco_1_1NotificationCenter}{Notification\+Center}} and an object receiving notifications from it. It is quite similar in concept to the \mbox{\hyperlink{classPoco_1_1RunnableAdapter}{Runnable\+Adapter}}, but provides some \mbox{\hyperlink{classPoco_1_1NotificationCenter}{Notification\+Center}} specific additional methods. See the \mbox{\hyperlink{classPoco_1_1NotificationCenter}{Notification\+Center}} class for information on how to use this template class.

This class template is quite similar to the \mbox{\hyperlink{classPoco_1_1Observer}{Observer}} class template. The only difference is that the \mbox{\hyperlink{classPoco_1_1NObserver}{N\+Observer}} expects the callback function to accept a const \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}}\& instead of a plain pointer as argument, thus simplifying memory management.
