\subsubsection*{template$<$class C, class P = C$\ast$$>$\newline
class Poco\+::\+Poolable\+Object\+Factory$<$ C, P $>$}

A \mbox{\hyperlink{classPoco_1_1PoolableObjectFactory}{Poolable\+Object\+Factory}} is responsible for creating and resetting objects managed by an \mbox{\hyperlink{classPoco_1_1ObjectPool}{Object\+Pool}}.

Together with an \mbox{\hyperlink{classPoco_1_1ObjectPool}{Object\+Pool}}, a \mbox{\hyperlink{classPoco_1_1PoolableObjectFactory}{Poolable\+Object\+Factory}} is used as a policy class to change the behavior of the \mbox{\hyperlink{classPoco_1_1ObjectPool}{Object\+Pool}} when creating new objects, returning used objects back to the pool and destroying objects, when the pool itself is destroyed or shrinked.
