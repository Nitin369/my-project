\doxysubsection{Detailed Description}
The abstract base class for Abstract\+H\+T\+T\+P\+Request\+Handlers created by \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServer}{H\+T\+T\+P\+Server}}.

Derived classes must override the \mbox{\hyperlink{classPoco_1_1Net_1_1AbstractHTTPRequestHandler_a348ba4cbaf51bdedbdafcc31d6ce22c6}{run()}} method. Contrary to a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler}{H\+T\+T\+P\+Request\+Handler}}, an \mbox{\hyperlink{classPoco_1_1Net_1_1AbstractHTTPRequestHandler}{Abstract\+H\+T\+T\+P\+Request\+Handler}} stores request and response as member variables to avoid having to pass them around as method parameters. Additionally, a \mbox{\hyperlink{classPoco_1_1Net_1_1HTMLForm}{H\+T\+M\+L\+Form}} object is created for use by subclasses.

The \mbox{\hyperlink{classPoco_1_1Net_1_1AbstractHTTPRequestHandler_a348ba4cbaf51bdedbdafcc31d6ce22c6}{run()}} method must perform the complete handling of the H\+T\+TP request connection. As soon as the \mbox{\hyperlink{classPoco_1_1Net_1_1AbstractHTTPRequestHandler_a348ba4cbaf51bdedbdafcc31d6ce22c6}{run()}} method returns, the request handler object is destroyed.

A new \mbox{\hyperlink{classPoco_1_1Net_1_1AbstractHTTPRequestHandler}{Abstract\+H\+T\+T\+P\+Request\+Handler}} object will be created for each new H\+T\+TP request that is received by the \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServer}{H\+T\+T\+P\+Server}}.
