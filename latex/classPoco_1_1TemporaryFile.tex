\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1TemporaryFile}{Temporary\+File}} class helps with the handling of temporary files. A unique name for the temporary file is automatically chosen and the file is placed in the directory reserved for temporary files (see \mbox{\hyperlink{classPoco_1_1Path_a741af3dc6b4a6b2791ec94a73df89176}{Path\+::temp()}}). Obtain the path by calling the \mbox{\hyperlink{classPoco_1_1File_aa50604d122698337ab3c1aadc41b3b08}{path()}} method (inherited from \mbox{\hyperlink{classPoco_1_1File}{File}}).

The \mbox{\hyperlink{classPoco_1_1TemporaryFile}{Temporary\+File}} class does not actually create the file -\/ this is up to the application. The class does, however, delete the temporary file -\/ either in the destructor, or immediately before the application terminates.
