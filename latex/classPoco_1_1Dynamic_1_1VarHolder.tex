\doxysubsection{Detailed Description}
Interface for a data holder used by the \mbox{\hyperlink{classPoco_1_1Dynamic_1_1Var}{Var}} class. Provides methods to convert between data types. Only data types for which \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolder}{Var\+Holder}} specialization exists are supported.

Provided are specializations for all C++ built-\/in types with addition of std\+::string, \mbox{\hyperlink{namespacePoco_a48326c5073e372134caf2558e2c5b531}{Poco\+::\+U\+T\+F16\+String}}, \mbox{\hyperlink{classPoco_1_1DateTime}{Date\+Time}}, \mbox{\hyperlink{classPoco_1_1LocalDateTime}{Local\+Date\+Time}}, \mbox{\hyperlink{classPoco_1_1Timestamp}{Timestamp}}, std\+::vector$<$\+Var$>$ and Dynamic\+Struct.

Additional types can be supported by adding specializations. When implementing specializations, the only condition is that they reside in \mbox{\hyperlink{namespacePoco}{Poco}} namespace and implement the pure virtual functions \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolder_ab96c9c02166f0bb0d0e4c79d64a13e28}{clone()}} and \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolder_a268bd432f87c533850c03422427b484b}{type()}}.

Those conversions that are not implemented shall fail back to this base class implementation. All the \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolder_aff77778b8b597d2b003b059093a0457e}{convert()}} function overloads in this class throw Bad\+Cast\+Exception.
