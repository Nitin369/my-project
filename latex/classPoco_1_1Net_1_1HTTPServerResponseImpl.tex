\doxysubsection{Detailed Description}
This subclass of \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServerResponse}{H\+T\+T\+P\+Server\+Response}} is used for representing server-\/side H\+T\+TP responses.

A \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServerResponse}{H\+T\+T\+P\+Server\+Response}} is passed to the handle\+Request() method of \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler}{H\+T\+T\+P\+Request\+Handler}}.

handle\+Request() must set a status code and optional reason phrase, set headers as necessary, and provide a message body.
