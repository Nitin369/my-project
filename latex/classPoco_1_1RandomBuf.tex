\doxysubsection{Detailed Description}
This streambuf generates random data. On Windows NT, the cryptographic A\+PI is used. On Unix, /dev/random is used, if available. Otherwise, a random number generator, some more-\/or-\/less random data and a S\+H\+A-\/1 digest is used to generate random data.
