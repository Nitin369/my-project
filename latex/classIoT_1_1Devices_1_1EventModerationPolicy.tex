class Io\+T\+::\+Devices\+::\+Event\+Moderation\+Policy$<$ T $>$}

An \mbox{\hyperlink{classIoT_1_1Devices_1_1EventModerationPolicy}{Event\+Moderation\+Policy}} controls the frequency of events sent when an evented sensor or state variable changes. Any class or class template that fulfills the following criteria can be used as \mbox{\hyperlink{classIoT_1_1Devices_1_1EventModerationPolicy}{Event\+Moderation\+Policy}}\+:
\begin{DoxyItemize}
\item The class must provide a public member function named value\+Changed that takes as only argument the new value of the state variable (passed by const reference).
\item \mbox{\hyperlink{classIoT_1_1Devices_1_1EventModerationPolicy_aebcc5ea74927f383aedb6cb2d18ad7fc}{value\+Changed()}} must check whether the criteria for firing an event are fulfilled, and in if this is the case, fire the event.
\item Typically, the \mbox{\hyperlink{classPoco_1_1BasicEvent}{Poco\+::\+Basic\+Event}} instance will be passed to the policy object through the constructor. 
\end{DoxyItemize}
