Creates the \mbox{\hyperlink{classPoco_1_1Zip_1_1PartialOutputStream}{Partial\+Output\+Stream}} and connects it to the given output stream. Bytes written are guaranteed to be in the range \mbox{[}start, real\+End -\/ end\mbox{]}. If init\+Stream is true the status of the stream will be cleared on the first access. start and end acts as offset values for the written content. A start value greater than zero, means that the first bytes are not written but discarded instead, an end value not equal to zero means that the last end bytes are not written! Examples\+: start = 3; end = 1 write(\char`\"{}hello\char`\"{}, 5) -\/$>$ \char`\"{}l\char`\"{}
