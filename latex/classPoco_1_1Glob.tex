\doxysubsection{Detailed Description}
This class implements glob-\/style pattern matching as known from Unix shells.

In the pattern string, \textquotesingle{}$\ast$\textquotesingle{} matches any sequence of characters, \textquotesingle{}?\textquotesingle{} matches any single character, \mbox{[}S\+ET\mbox{]} matches any single character in the specified set, \mbox{[}!\+S\+ET\mbox{]} matches any character not in the ~\newline
 specified set.

A set is composed of characters or ranges; a range looks like character hyphen character (as in 0-\/9 or A-\/Z). \mbox{[}0-\/9a-\/z\+A-\/\+Z\+\_\+\mbox{]} is the set of characters allowed in C identifiers. \mbox{\hyperlink{classPoco_1_1Any}{Any}} other character in the pattern must be matched exactly.

To suppress the special syntactic significance of any of \textquotesingle{}\mbox{[}\mbox{]}$\ast$?!-\/\textbackslash{}\textquotesingle{}, and match the character exactly, precede it with a backslash.

All strings are assumed to be U\+T\+F-\/8 encoded.
