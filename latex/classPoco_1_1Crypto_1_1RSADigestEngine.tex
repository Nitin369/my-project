\doxysubsection{Detailed Description}
This class implements a \mbox{\hyperlink{classPoco_1_1DigestEngine}{Poco\+::\+Digest\+Engine}} that can be used to compute a secure digital signature.

First another \mbox{\hyperlink{classPoco_1_1Crypto_1_1DigestEngine}{Poco\+::\+Crypto\+::\+Digest\+Engine}} is created and used to compute a cryptographic hash of the data to be signed. Then, the hash value is encrypted, using the R\+SA private key.

To verify a signature, pass it to the \mbox{\hyperlink{classPoco_1_1Crypto_1_1RSADigestEngine_acc3ccf1b489369a48e84708e4a5f7794}{verify()}} member function. It will decrypt the signature using the R\+SA public key and compare the resulting hash with the actual hash of the data.
