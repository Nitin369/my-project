A factory for \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPClientSession}{H\+T\+T\+P\+Client\+Session}} objects. \begin{DoxyVerb}Creates a HTTP session for a given URI.
A HTTPSessionInstantiator is not used directly.
Instances are registered with a HTTPSessionFactory,
and used through it.
