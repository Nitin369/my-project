\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1EventTarget}{Event\+Target}} interface is implemented by all Nodes in an implementation which supports the D\+OM \mbox{\hyperlink{classPoco_1_1XML_1_1Event}{Event}} Model. Therefore, this interface can be obtained by using binding-\/specific casting methods on an instance of the \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}} interface. The interface allows registration and removal of Event\+Listeners on an \mbox{\hyperlink{classPoco_1_1XML_1_1EventTarget}{Event\+Target}} and dispatch of events to that \mbox{\hyperlink{classPoco_1_1XML_1_1EventTarget}{Event\+Target}}.
