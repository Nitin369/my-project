\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}} (mutual exclusion) is a synchronization mechanism used to control access to a shared resource in a concurrent (multithreaded) scenario. Mutexes are recursive, that is, the same mutex can be locked multiple times by the same thread (but, of course, not by other threads). Using the \mbox{\hyperlink{classPoco_1_1ScopedLock}{Scoped\+Lock}} class is the preferred way to automatically lock and unlock a mutex.
