\doxysubsection{Detailed Description}
A better random number generator. \mbox{\hyperlink{classPoco_1_1Random}{Random}} implements a pseudo random number generator (P\+R\+NG). The P\+R\+NG is a nonlinear additive feedback random number generator using 256 bytes of state information and a period of up to 2$^\wedge$69.
