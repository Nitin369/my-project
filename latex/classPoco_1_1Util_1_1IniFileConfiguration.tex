\doxysubsection{Detailed Description}
This implementation of a Configuration reads properties from a legacy Windows initialization (.ini) file.

The file syntax is implemented as follows.
\begin{DoxyItemize}
\item a line starting with a semicolon is treated as a comment and ignored
\item a line starting with a square bracket denotes a section key \mbox{[}$<$key$>$\mbox{]}
\item every other line denotes a property assignment in the form 
\end{DoxyItemize}

= 

The name of a property is composed of the section key and the value key, separated by a period ($<$section key$>$.

).

Property names are not case sensitive. Leading and trailing whitespace is removed from both keys and values.
