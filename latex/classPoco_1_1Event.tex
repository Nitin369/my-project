\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classPoco_1_1Event}{Event}} is a synchronization object that allows one thread to signal one or more other threads that a certain event has happened. Usually, one thread signals an event, while one or more other threads wait for an event to become signalled.
