\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1NodeList}{Node\+List}} interface provides the abstraction of an ordered collection of nodes, without defining or constraining how this collection is implemented.

The items in the \mbox{\hyperlink{classPoco_1_1XML_1_1NodeList}{Node\+List}} are accessible via an integral index, starting from 0.

A \mbox{\hyperlink{classPoco_1_1XML_1_1NodeList}{Node\+List}} returned from a method must be released with a call to \mbox{\hyperlink{classPoco_1_1XML_1_1DOMObject_a59fcc225a5bcf0fcf7e49d06ecba249a}{release()}} when no longer needed.
