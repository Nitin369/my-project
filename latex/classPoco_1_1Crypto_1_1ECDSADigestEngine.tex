\doxysubsection{Detailed Description}
This class implements a \mbox{\hyperlink{classPoco_1_1DigestEngine}{Poco\+::\+Digest\+Engine}} that can be used to compute a secure digital signature.

First another \mbox{\hyperlink{classPoco_1_1Crypto_1_1DigestEngine}{Poco\+::\+Crypto\+::\+Digest\+Engine}} is created and used to compute a cryptographic hash of the data to be signed. Then, the hash value is encrypted, using the E\+C\+D\+SA private key.

To verify a signature, pass it to the \mbox{\hyperlink{classPoco_1_1Crypto_1_1ECDSADigestEngine_ada34e71f623b75306bf5520735bf7fa4}{verify()}} member function. It will decrypt the signature using the E\+C\+D\+SA public key and compare the resulting hash with the actual hash of the data.
