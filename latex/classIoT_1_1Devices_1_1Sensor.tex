\doxysubsection{Detailed Description}
The base class for analog sensors, such as temperature or ambient light sensors.

In addition to the methods defined in this interface, a \mbox{\hyperlink{classIoT_1_1Devices_1_1Sensor}{Sensor}} implementation should expose the following properties\+:
\begin{DoxyItemize}
\item physical\+Quantity (string)\+: The physical quantity that is being measured by the sensor, e.\+g. \char`\"{}temperature\char`\"{}.
\item physical\+Unit (string)\+: The physical unit the measured value is being represented in (e.\+g. \char`\"{}Â°\+C\char`\"{}), U\+T\+F-\/8 encoded. See the P\+H\+Y\+S\+I\+C\+A\+L\+\_\+\+U\+N\+I\+T\+\_\+$\ast$ strings for predefined values.
\item display\+Value (string, optional)\+: The current value of the sensor, formatted as string for display purposes. 
\end{DoxyItemize}
