\doxysubsection{Detailed Description}
A task that can be scheduled for one-\/time or repeated execution by a \mbox{\hyperlink{classPoco_1_1Util_1_1Timer}{Timer}}.

This is an abstract class. Subclasses must override the \mbox{\hyperlink{classPoco_1_1Runnable_ad89816e81207bf5f1e52d5df860d11d4}{run()}} member function to implement the actual task logic.

{\footnotesize\ttfamily bool Poco\+::\+Util\+::\+Timer\+Task\+::is\+Cancelled (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [inline]}}

Cancels the execution of the timer. If the task has been scheduled for one-\/time execution and has not yet run, or has not yet been scheduled, it will never run. If the task has been scheduled for repeated execution, it will never run again. If the task is running when this call occurs, the task will run to completion, but will never run again.

Warning\+: A \mbox{\hyperlink{classPoco_1_1Util_1_1TimerTask}{Timer\+Task}} that has been cancelled must not be scheduled again. An attempt to do so results in a Poco\+::\+Util\+::\+Illegal\+State\+Exception being thrown.
