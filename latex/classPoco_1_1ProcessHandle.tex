\doxysubsection{Detailed Description}
A handle for a process created with \mbox{\hyperlink{classPoco_1_1Process_a2dd969e0d5d36b7d0600a088d68ea3dc}{Process\+::launch()}}. \begin{DoxyVerb}This handle can be used to determine the process ID of
the newly created process and it can be used to wait for
the completion of a process. 
\end{DoxyVerb}
