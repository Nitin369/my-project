Creates and opens a H\+T\+TP stream for the given \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}. The \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} must be a \href{http://..}{\texttt{ http\+://..}}. \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}.

Throws a Net\+Exception if anything goes wrong.

Redirect responses are handled and the redirect location is automatically resolved, as long as the redirect location is still accessible via the H\+T\+TP protocol. If a redirection to a non \href{http://..}{\texttt{ http\+://..}}. \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} is received, a Unsupported\+Redirect\+Exception exception is thrown.
