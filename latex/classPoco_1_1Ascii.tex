\doxysubsection{Detailed Description}
This class contains enumerations and static utility functions for dealing with A\+S\+C\+II characters and their properties.

The classification functions will also work if non-\/\+A\+S\+C\+II character codes are passed to them, but classification will only check for A\+S\+C\+II characters.

This allows the classification methods to be used on the single bytes of a U\+T\+F-\/8 string, without causing assertions or inconsistent results (depending upon the current locale) on bytes outside the A\+S\+C\+II range, as may be produced by \mbox{\hyperlink{classPoco_1_1Ascii_af972424d47bf76090f82edb8235cf293}{Ascii\+::is\+Space()}}, etc.
