This class provides a default implementation of the S\+A\+X2 \mbox{\hyperlink{classPoco_1_1XML_1_1Attributes}{Attributes}} interface, with the addition of manipulators so that the list can be modified or reused.

There are two typical uses of this class\+:
\begin{DoxyEnumerate}
\item to take a persistent snapshot of an \mbox{\hyperlink{classPoco_1_1XML_1_1Attributes}{Attributes}} object in a start\+Element event; or
\item to construct or modify an \mbox{\hyperlink{classPoco_1_1XML_1_1Attributes}{Attributes}} object in a S\+A\+X2 driver or filter. 
\end{DoxyEnumerate}
