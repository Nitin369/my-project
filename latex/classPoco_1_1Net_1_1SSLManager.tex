\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager}{S\+S\+L\+Manager}} is a singleton for holding the default server/client \mbox{\hyperlink{classPoco_1_1Net_1_1Context}{Context}} and handling callbacks for certificate verification errors and private key passphrases.

Proper initialization of \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager}{S\+S\+L\+Manager}} is critical.

\mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager}{S\+S\+L\+Manager}} can be initialized manually, by calling \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_a524db55bb65600424b2eaa016f98852b}{initialize\+Server()}} and/or \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_a0aef1a1df619765075ec7d2925aaa68b}{initialize\+Client()}}, or initialization can be automatic. In the latter case, a \mbox{\hyperlink{classPoco_1_1Util_1_1Application}{Poco\+::\+Util\+::\+Application}} instance must be available and the required configuration properties must be set (see below).

Note that manual initialization must happen very early in the application, before \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_a78c9f84e5b9836609838777f6d04f524}{default\+Client\+Context()}} or \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_ae302e2245d821a766f4564c0882ba53e}{default\+Server\+Context()}} are called.

If \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_a78c9f84e5b9836609838777f6d04f524}{default\+Client\+Context()}} and \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager_ae302e2245d821a766f4564c0882ba53e}{default\+Server\+Context()}} are never called in an application, initialization of \mbox{\hyperlink{classPoco_1_1Net_1_1SSLManager}{S\+S\+L\+Manager}} can be omitted. However, in this case, delegates for the Server\+Verification\+Error, Client\+Verification\+Error and Private\+Key\+Passphrase\+Required events must be registered.

An exemplary documentation which sets either the server or client default context and creates a \mbox{\hyperlink{classPoco_1_1Net_1_1PrivateKeyPassphraseHandler}{Private\+Key\+Passphrase\+Handler}} that reads the password from the \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} file looks like this\+:

Following is a list of supported configuration properties. Property names must always be prefixed with open\+S\+S\+L.\+server or open\+S\+S\+L.\+client. Some properties are only supported for servers.


\begin{DoxyItemize}
\item private\+Key\+File (string)\+: The path to the file containing the private key for the certificate in P\+EM format (or containing both the private key and the certificate).
\item certificate\+File (string)\+: The \mbox{\hyperlink{classPoco_1_1Path}{Path}} to the file containing the server\textquotesingle{}s or client\textquotesingle{}s certificate in P\+EM format. Can be omitted if the the file given in private\+Key\+File contains the certificate as well.
\item ca\+Config (string)\+: The path to the file or directory containing the trusted root certificates.
\item verification\+Mode (string)\+: Specifies whether and how peer certificates are validated (see the \mbox{\hyperlink{classPoco_1_1Net_1_1Context}{Context}} class for details). Valid values are none, relaxed, strict, once.
\item verification\+Depth (integer, 1-\/9)\+: Sets the upper limit for verification chain sizes. Verification will fail if a certificate chain larger than this is encountered.
\item load\+Default\+C\+A\+File (boolean)\+: Specifies whether the builtin CA certificates from Open\+S\+SL are used.
\item cipher\+List (string)\+: Specifies the supported ciphers in Open\+S\+SL notation (e.\+g. \char`\"{}\+A\+L\+L\+:!\+A\+D\+H\+:!\+L\+O\+W\+:!\+E\+X\+P\+:!\+M\+D5\+:@\+S\+T\+R\+E\+N\+G\+T\+H\char`\"{}).
\item prefer\+Server\+Ciphers (bool)\+: When choosing a cipher, use the server\textquotesingle{}s preferences instead of the client preferences. When not called, the S\+SL server will always follow the clients preferences. When called, the S\+S\+L/\+T\+LS server will choose following its own preferences.
\item private\+Key\+Passphrase\+Handler.\+name (string)\+: The name of the class (subclass of \mbox{\hyperlink{classPoco_1_1Net_1_1PrivateKeyPassphraseHandler}{Private\+Key\+Passphrase\+Handler}}) used for obtaining the passphrase for accessing the private key.
\item private\+Key\+Passphrase\+Handler.\+options.\+password (string)\+: The password to be used by \mbox{\hyperlink{classPoco_1_1Net_1_1KeyFileHandler}{Key\+File\+Handler}}.
\item invalid\+Certificate\+Handler.\+name\+: The name of the class (subclass of Certificate\+Handler) used for confirming invalid certificates.
\item cache\+Sessions (boolean)\+: Enables or disables session caching.
\item session\+Id\+Context (string)\+: contains the application\textquotesingle{}s unique session ID context, which becomes part of each session identifier generated by the server. Can be an arbitrary sequence of bytes with a maximum length of S\+S\+L\+\_\+\+M\+A\+X\+\_\+\+S\+S\+L\+\_\+\+S\+E\+S\+S\+I\+O\+N\+\_\+\+I\+D\+\_\+\+L\+E\+N\+G\+TH. If not specified, defaults to \$\{application.\+name\}.
\item session\+Cache\+Size (integer)\+: Sets the maximum size of the server session cache, in number of sessions. The default size (according to Open\+S\+SL documentation) is 1024$\ast$20, which may be too large for many applications, especially on embedded platforms with limited memory. Specifying a size of 0 will set an unlimited cache size.
\item session\+Timeout (integer)\+: Sets the timeout (in seconds) of cached sessions on the server.
\item extended\+Verification (boolean)\+: Enable or disable the automatic post-\/connection extended certificate verification.
\item require\+T\+L\+Sv1 (boolean)\+: Require a T\+L\+Sv1 connection.
\item require\+T\+L\+Sv1\+\_\+1 (boolean)\+: Require a T\+L\+Sv1.\+1 connection.
\item require\+T\+L\+Sv1\+\_\+2 (boolean)\+: Require a T\+L\+Sv1.\+2 connection.

\item dh\+Params\+File (string)\+: Specifies a file containing Diffie-\/\+Hellman parameters. If not specified or empty, the default parameters are used.
\item ecdh\+Curve (string)\+: Specifies the name of the curve to use for E\+C\+DH, based on the curve names specified in R\+FC 4492. Defaults to \char`\"{}prime256v1\char`\"{}.
