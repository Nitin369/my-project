\doxysubsection{Detailed Description}
This helper class manages event listener subscriptions and event dispatching for \mbox{\hyperlink{classPoco_1_1XML_1_1AbstractNode}{Abstract\+Node}}.

The \mbox{\hyperlink{classPoco_1_1XML_1_1EventListener}{Event\+Listener}} list is managed in such a way that event listeners can be added and removed even from within an \mbox{\hyperlink{classPoco_1_1XML_1_1EventListener}{Event\+Listener}}, while events are being dispatched.
