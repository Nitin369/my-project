\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleActivator}{Bundle\+Activator}} is the entry point for every bundle contributing executable code and requiring special action upon startup or shutdown.

A bundle should provide its own \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleActivator}{Bundle\+Activator}} class derived from this class.

The \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleActivator}{Bundle\+Activator}} (or, correctly speaking, a class derived from it) is the only class loaded from the bundle by the \mbox{\hyperlink{namespacePoco_1_1OSP}{O\+SP}} framework.

To prevent lengthy startup and shutdown sequences, the \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleActivator_a86ba6c4816e5b1e151449a0d41bf7ba5}{start()}} and \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleActivator_aa9dedcdab137f26891a8de1c9885501f}{stop()}} method should do their work in a timely fashion.
