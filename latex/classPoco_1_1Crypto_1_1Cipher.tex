\doxysubsection{Detailed Description}
Represents the abstract base class from which all implementations of symmetric/asymmetric encryption algorithms must inherit. Use the \mbox{\hyperlink{classPoco_1_1Crypto_1_1CipherFactory}{Cipher\+Factory}} class to obtain an instance of this class\+: \begin{DoxyVerb}CipherFactory& factory = CipherFactory::defaultFactory();
// Creates a 256-bit AES cipher
Cipher* pCipher = factory.createCipher(CipherKey("aes-256"));
Cipher* pRSACipher = factory.createCipher(RSAKey(RSAKey::KL_1024, RSAKey::EXP_SMALL));
\end{DoxyVerb}


Check the different Key constructors on how to initialize/create a key. The above example auto-\/generates random keys.

Note that you won\textquotesingle{}t be able to decrypt data encrypted with a random key once the \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher}{Cipher}} is destroyed unless you persist the generated key and IV. An example usage for random keys is to encrypt data saved in a temporary file.

Once your key is set up, you can use the \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher}{Cipher}} object to encrypt or decrypt strings or, in conjunction with a \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoInputStream}{Crypto\+Input\+Stream}} or a \mbox{\hyperlink{classPoco_1_1Crypto_1_1CryptoOutputStream}{Crypto\+Output\+Stream}}, to encrypt streams of data.

Since encrypted strings will contain arbitrary binary data that will cause problems in applications that are not binary-\/safe (eg., when sending encrypted data in e-\/mails), the \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher_a8effae971627f5a47245c2558035bc64}{encrypt\+String()}} and \mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher_a813880d0d88dad75620910caa3341543}{decrypt\+String()}} can encode (or decode, respectively) encrypted data using a \char`\"{}transport encoding\char`\"{}. Supported encodings are Base64 and Bin\+Hex.

The following example encrypts and decrypts a string utilizing Base64 encoding\+: \begin{DoxyVerb}std::string plainText = "This is my secret information";
std::string encrypted = pCipher->encryptString(plainText, Cipher::ENC_BASE64);
std::string decrypted = pCipher->decryptString(encrypted, Cipher::ENC_BASE64);
\end{DoxyVerb}


In order to encrypt a stream of data (eg. to encrypt files), you can use a Crypto\+Stream\+: \begin{DoxyVerb}// Create an output stream that will encrypt all data going through it
// and write pass it to the underlying file stream.
Poco::FileOutputStream sink("encrypted.dat");
CryptoOutputStream encryptor(sink, pCipher->createEncryptor());

Poco::FileInputStream source("source.txt");
Poco::StreamCopier::copyStream(source, encryptor);

// Always close output streams to flush all internal buffers
encryptor.close();
sink.close(); 
\end{DoxyVerb}
