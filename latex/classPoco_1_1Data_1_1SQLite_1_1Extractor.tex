{\footnotesize\ttfamily void Poco\+::\+Data\+::\+S\+Q\+Lite\+::\+Extractor\+::reset (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}, {\ttfamily [virtual]}}

Returns true if the current row value at pos column is null. Because of the loss of information about null-\/ness of the underlying database values due to the nature of \mbox{\hyperlink{namespacePoco_1_1Data_1_1SQLite}{S\+Q\+Lite}} engine, (once null value is converted to default value, \mbox{\hyperlink{namespacePoco_1_1Data_1_1SQLite}{S\+Q\+Lite}} A\+PI treats it as non-\/null), a null indicator container member variable is used to cache the indicators of the underlying nulls thus rendering this function idempotent. The container is a vector of \mbox{[}bool, bool\mbox{]} pairs. The vector index corresponds to the column position, the first bool value in the pair is true if the null indicator has been set and the second bool value in the pair is true if the column is actually null. The row argument, needed for connectors with bulk capabilities, is ignored in this implementation.
