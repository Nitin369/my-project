\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1NodeAppender}{Node\+Appender}} class provides a very fast way to build larger D\+OM documents.

In the D\+OM, child nodes are usually appended to a parent node using the \mbox{\hyperlink{classPoco_1_1XML_1_1NodeAppender_a9995937be5e35f3831e1993fab08348a}{append\+Child()}} method. For nodes containing more than a few children, this method can be quite slow, due to the way it\textquotesingle{}s implemented, and because of the requirements of the D\+OM specification.

While the \mbox{\hyperlink{classPoco_1_1XML_1_1NodeAppender}{Node\+Appender}} is being used on an \mbox{\hyperlink{classPoco_1_1XML_1_1Element}{Element}}, no children-\/modifying methods of that \mbox{\hyperlink{classPoco_1_1XML_1_1Element}{Element}} must be used.

This class is not part of the D\+OM specification.
