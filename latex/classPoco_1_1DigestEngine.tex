\doxysubsection{Detailed Description}
This class is an abstract base class for all classes implementing a message digest algorithm, like \mbox{\hyperlink{classPoco_1_1MD5Engine}{M\+D5\+Engine}} and \mbox{\hyperlink{classPoco_1_1SHA1Engine}{S\+H\+A1\+Engine}}. Call \mbox{\hyperlink{classPoco_1_1DigestEngine_ac3de0beba3214e7c41b97ceab478f4d1}{update()}} repeatedly with data to compute the digest from. When done, call \mbox{\hyperlink{classPoco_1_1DigestEngine_acbb5e07fd2f4ae327c3a6897867e07f3}{digest()}} to obtain the message digest.
