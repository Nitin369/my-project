\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Clock}{Clock}} stores a monotonic$\ast$ clock value with (theoretical) microseconds resolution. Clocks can be compared with each other and simple arithmetics are supported.

\mbox{[}$\ast$\mbox{]} Note that \mbox{\hyperlink{classPoco_1_1Clock}{Clock}} values are only monotonic if the operating system provides a monotonic clock. The \mbox{\hyperlink{classPoco_1_1Clock_adcc973fce2c226c21aaa9cf137310c11}{monotonic()}} function can be used to check whether the system\textquotesingle{}s clock is monotonic.

Monotonic \mbox{\hyperlink{classPoco_1_1Clock}{Clock}} is available on Windows, Linux, OS X and on P\+O\+S\+IX platforms supporting clock\+\_\+gettime() with C\+L\+O\+C\+K\+\_\+\+M\+O\+N\+O\+T\+O\+N\+IC. ~\newline


\mbox{\hyperlink{classPoco_1_1Clock}{Clock}} values are relative to a system-\/dependent epoch time (usually the system\textquotesingle{}s startup time) and have no relation to the time of day.
