\doxysubsection{Detailed Description}
This class encapsulates an H\+T\+TP request message.

In addition to the properties common to all H\+T\+TP messages, a H\+T\+TP request has a method (e.\+g. G\+ET, H\+E\+AD, P\+O\+ST, etc.) and a request \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}.
