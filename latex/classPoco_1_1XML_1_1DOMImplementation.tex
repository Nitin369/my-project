The \mbox{\hyperlink{classPoco_1_1XML_1_1DOMImplementation}{D\+O\+M\+Implementation}} interface provides a number of methods for performing operations that are independent of any particular instance of the document object model. In this implementation, \mbox{\hyperlink{classPoco_1_1XML_1_1DOMImplementation}{D\+O\+M\+Implementation}} is implemented as a singleton.
