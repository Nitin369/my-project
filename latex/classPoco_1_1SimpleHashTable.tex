\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Key, class Value, class Key\+Hash\+Function = Hash\+Function$<$\+Key$>$$>$\newline
class Poco\+::\+Simple\+Hash\+Table$<$ Key, Value, Key\+Hash\+Function $>$}

A \mbox{\hyperlink{classPoco_1_1SimpleHashTable}{Simple\+Hash\+Table}} stores a key value pair that can be looked up via a hashed key. \begin{DoxyVerb}In comparision to a HashTable, this class handles collisions by sequentially searching the next
free location. This also means that the maximum size of this table is limited, i.e. if the hash table
is full, it will throw an exception and that this class does not support remove operations.
On the plus side it is faster than the HashTable.

This class is NOT thread safe. 
\end{DoxyVerb}
