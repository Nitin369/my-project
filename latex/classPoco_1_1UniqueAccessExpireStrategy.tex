\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T\+Key, class T\+Value$>$\newline
class Poco\+::\+Unique\+Access\+Expire\+Strategy$<$ T\+Key, T\+Value $>$}

An \mbox{\hyperlink{classPoco_1_1UniqueExpireStrategy}{Unique\+Expire\+Strategy}} implements time based expiration of cache entries. In contrast to \mbox{\hyperlink{classPoco_1_1ExpireStrategy}{Expire\+Strategy}} which only allows to set a per cache expiration value, it allows to define expiration per Cache\+Entry. Each T\+Value object must thus offer the following method\+:

const \mbox{\hyperlink{classPoco_1_1Timestamp}{Poco\+::\+Timestamp}}\& get\+Timeout() const;

which returns the timespan for how long an object will be valid without being accessed.
