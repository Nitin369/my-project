\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Data_1_1SQLite_1_1Notifier}{Notifier}} is a wrapper for \mbox{\hyperlink{namespacePoco_1_1Data_1_1SQLite}{S\+Q\+Lite}} callback calls. It supports event callbacks for insert, update, delete, commit and rollback events. While (un)registering callbacks is thread-\/safe, execution of the callbacks themselves are not; it is the user\textquotesingle{}s responsibility to ensure the thread-\/safey of the functions they provide as callback target. Additionally, commit callbacks may prevent database transactions from succeeding (see sqlite\+Commit\+Callback\+Fn documentation for details).

There can be only one set of callbacks per session (i.\+e. registering a new callback automatically unregisters the previous one). All callbacks are registered and enabled at \mbox{\hyperlink{classPoco_1_1Data_1_1SQLite_1_1Notifier}{Notifier}} contruction time and can be disabled at a later point time.
