\doxysubsection{Detailed Description}
A convenience class that converts strings from U\+T\+F-\/8 encoded std\+::strings to U\+T\+F-\/16 or U\+T\+F-\/32 encoded std\+::wstrings and vice-\/versa.

This class is mainly used for working with the \mbox{\hyperlink{classPoco_1_1Unicode}{Unicode}} Windows A\+P\+Is and probably won\textquotesingle{}t be of much use anywhere else ???
