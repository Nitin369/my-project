\doxysubsection{Detailed Description}
This class defines the interface that all \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} stream factories must implement.

Subclasses must implement the \mbox{\hyperlink{classPoco_1_1URIStreamFactory_a0768071cef5d26530977d4fd23a8abc8}{open()}} method.
