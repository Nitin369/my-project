\doxysubsection{Detailed Description}
An abstract base class for implementing text encodings like U\+T\+F-\/8 or I\+SO 8859-\/1.

Subclasses must override the \mbox{\hyperlink{classPoco_1_1TextEncoding_a128efbeb06e5b102704633d3d8565ea3}{canonical\+Name()}}, \mbox{\hyperlink{classPoco_1_1TextEncoding_aa199506fe71af0bb8bef4b735880ba48}{is\+A()}}, \mbox{\hyperlink{classPoco_1_1TextEncoding_a7c3e03c076147725384737ec0a4471ae}{character\+Map()}} and \mbox{\hyperlink{classPoco_1_1TextEncoding_a318ff7d0d0dfc8419d2358bc68145531}{convert()}} methods and need to be thread safe and stateless.

\mbox{\hyperlink{classPoco_1_1TextEncoding}{Text\+Encoding}} also provides static member functions for managing mappings from encoding names to \mbox{\hyperlink{classPoco_1_1TextEncoding}{Text\+Encoding}} objects.

The map\mbox{[}b\mbox{]} member gives information about byte sequences whose first byte is b. If map\mbox{[}b\mbox{]} is c where c is $>$= 0, then b by itself encodes the \mbox{\hyperlink{classPoco_1_1Unicode}{Unicode}} scalar value c. If map\mbox{[}b\mbox{]} is -\/1, then the byte sequence is malformed. If map\mbox{[}b\mbox{]} is -\/n, where n $>$= 2, then b is the first byte of an n-\/byte sequence that encodes a single \mbox{\hyperlink{classPoco_1_1Unicode}{Unicode}} scalar value. Byte sequences up to 6 bytes in length are supported.

{\footnotesize\ttfamily virtual int Poco\+::\+Text\+Encoding\+::convert (\begin{DoxyParamCaption}\item[{int}]{ch,  }\item[{unsigned char $\ast$}]{bytes,  }\item[{int}]{length }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [virtual]}}

The sequence\+Length function is used to get the lenth of the sequence pointed by bytes. The length paramater should be greater or equal to the length of the sequence.

The sequence\+Length function must return the lenght of the sequence represented by this byte sequence or a negative value -\/n if length is shorter than the sequence, where n is the number of byte requested to determine the length of the sequence.
