\doxysubsection{Detailed Description}
This \mbox{\hyperlink{classPoco_1_1Formatter}{Formatter}} allows for custom formatting of log messages based on format patterns.

The format pattern is used as a template to format the message and is copied character by character except for the following special characters, which are replaced by the corresponding value.


\begin{DoxyItemize}
\item s -\/ message source
\item t -\/ message text
\item l -\/ message priority level (1 .. 7)
\item p -\/ message priority (Fatal, Critical, \mbox{\hyperlink{classPoco_1_1Error}{Error}}, Warning, Notice, Information, Debug, Trace)
\item q -\/ abbreviated message priority (F, C, E, W, N, I, D, T)
\item P -\/ message process identifier
\item T -\/ message thread name
\item I -\/ message thread identifier (numeric)
\item N -\/ node or host name
\item U -\/ message source file path (empty string if not set)
\item u -\/ message source line number (0 if not set)
\item w -\/ message date/time abbreviated weekday (Mon, Tue, ...)
\item W -\/ message date/time full weekday (Monday, Tuesday, ...)
\item b -\/ message date/time abbreviated month (Jan, Feb, ...)
\item B -\/ message date/time full month (January, February, ...)
\item d -\/ message date/time zero-\/padded day of month (01 .. 31)
\item e -\/ message date/time day of month (1 .. 31)
\item f -\/ message date/time space-\/padded day of month ( 1 .. 31)
\item m -\/ message date/time zero-\/padded month (01 .. 12)
\item n -\/ message date/time month (1 .. 12)
\item o -\/ message date/time space-\/padded month ( 1 .. 12)
\item y -\/ message date/time year without century (70)
\item Y -\/ message date/time year with century (1970)
\item H -\/ message date/time hour (00 .. 23)
\item h -\/ message date/time hour (00 .. 12)
\item a -\/ message date/time am/pm
\item A -\/ message date/time A\+M/\+PM
\item M -\/ message date/time minute (00 .. 59)
\item S -\/ message date/time second (00 .. 59)
\item i -\/ message date/time millisecond (000 .. 999)
\item c -\/ message date/time centisecond (0 .. 9)
\item F -\/ message date/time fractional seconds/microseconds (000000 -\/ 999999)
\item z -\/ time zone differential in I\+SO 8601 format (Z or +N\+N.\+NN)
\item Z -\/ time zone differential in R\+FC format (G\+MT or +N\+N\+NN)
\item L -\/ convert time to local time (must be specified before any date/time specifier; does not itself output anything)
\item E -\/ epoch time (U\+TC, seconds since midnight, January 1, 1970)
\item v\mbox{[}width\mbox{]} -\/ the message source (s) but text length is padded/cropped to \textquotesingle{}width\textquotesingle{}
\item \%\mbox{[}name\mbox{]} -\/ the value of the message parameter with the given name
\item \%\% -\/ percent sign 
\end{DoxyItemize}
