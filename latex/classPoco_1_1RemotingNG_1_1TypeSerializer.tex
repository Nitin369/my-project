\mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeSerializer}{Type\+Serializer}} is used to serialize the content of a variable for transportation over the network.

The default \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeSerializer}{Type\+Serializer}} template simply forwards \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeSerializer_a68e1b45755caaac6758c781ad891a0ee}{serialize()}} calls to a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Serializer}{Serializer}}. This means that serializaation using the default template only works for types directly supported by \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Serializer}{Serializer}}.

Serialization of complex types can be done by providing specializations of the \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeSerializer}{Type\+Serializer}} class template.
