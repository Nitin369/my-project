Initialize the \mbox{\hyperlink{namespacePoco_1_1Crypto}{Crypto}} library, as well as the underlying Open\+S\+SL libraries, by calling \mbox{\hyperlink{classPoco_1_1Crypto_1_1OpenSSLInitializer_a250dc93509019eb12337f320db39da8c}{Open\+S\+S\+L\+Initializer\+::initialize()}}.

Should be called before using any class from the \mbox{\hyperlink{namespacePoco_1_1Crypto}{Crypto}} library. The \mbox{\hyperlink{namespacePoco_1_1Crypto}{Crypto}} library will be initialized automatically, through \mbox{\hyperlink{classPoco_1_1Crypto_1_1OpenSSLInitializer}{Open\+S\+S\+L\+Initializer}} instances held by various \mbox{\hyperlink{namespacePoco_1_1Crypto}{Crypto}} classes (\mbox{\hyperlink{classPoco_1_1Crypto_1_1Cipher}{Cipher}}, \mbox{\hyperlink{classPoco_1_1Crypto_1_1CipherKey}{Cipher\+Key}}, \mbox{\hyperlink{classPoco_1_1Crypto_1_1RSAKey}{R\+S\+A\+Key}}, \mbox{\hyperlink{classPoco_1_1Crypto_1_1X509Certificate}{X509\+Certificate}}). However, it is recommended to call \mbox{\hyperlink{namespacePoco_1_1Crypto_ad8f360a953be401036d7773f2b486a88}{initialize\+Crypto()}} in any case at application startup.

Can be called multiple times; however, for every call to \mbox{\hyperlink{namespacePoco_1_1Crypto_ad8f360a953be401036d7773f2b486a88}{initialize\+Crypto()}}, a matching call to \mbox{\hyperlink{namespacePoco_1_1Crypto_a89b45246856d85e8fead3524141e93f5}{uninitialize\+Crypto()}} must be performed.
