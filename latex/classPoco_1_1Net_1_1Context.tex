\doxysubsection{Detailed Description}
This class encapsulates context information for an S\+SL server or client, such as the certificate verification mode and the location of certificates and private key files, as well as the list of supported ciphers.

The \mbox{\hyperlink{classPoco_1_1Net_1_1Context}{Context}} class is also used to control S\+SL session caching on the server and client side.
