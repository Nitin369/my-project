\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}} is used to execute S\+QL statements. It does not contain code of its own. Its main purpose is to forward calls to the concrete \mbox{\hyperlink{classPoco_1_1Data_1_1StatementImpl}{Statement\+Impl}} stored inside. \mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}} execution can be synchronous or asynchronous. Synchronous ececution is achieved through \mbox{\hyperlink{classPoco_1_1Data_1_1Statement_a2be38d265a5c47b8210f15e4082eec4e}{execute()}} call, while asynchronous is achieved through \mbox{\hyperlink{classPoco_1_1Data_1_1Statement_a5bcf1fe15413e942c275c8d97e761c84}{execute\+Async()}} method call. An asynchronously executing statement should not be copied during the execution.

Note\+:

Once set as asynchronous through \textquotesingle{}async\textquotesingle{} manipulator, statement remains asynchronous for all subsequent execution calls, both \mbox{\hyperlink{classPoco_1_1Data_1_1Statement_a2be38d265a5c47b8210f15e4082eec4e}{execute()}} and \mbox{\hyperlink{classPoco_1_1Data_1_1Statement_a5bcf1fe15413e942c275c8d97e761c84}{execute\+Async()}}. However, calling execut\+Async() on a synchronous statement shall execute asynchronously but without altering the underlying statement\textquotesingle{}s synchronous nature.

Once asynchronous, a statement can be reverted back to synchronous state in two ways\+:

1) By calling set\+Async(false) 2) By means of \textquotesingle{}sync\textquotesingle{} or \textquotesingle{}reset\textquotesingle{} manipulators

See individual functions documentation for more details.

\mbox{\hyperlink{classPoco_1_1Data_1_1Statement}{Statement}} owns the \mbox{\hyperlink{classPoco_1_1Data_1_1RowFormatter}{Row\+Formatter}}, which can be provided externaly through set\+Formatter() member function. If no formatter is externally supplied to the statement, the \mbox{\hyperlink{classPoco_1_1Data_1_1SimpleRowFormatter}{Simple\+Row\+Formatter}} is lazy created and used.
