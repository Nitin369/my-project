\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Proxy}{Proxy}} maintains a connection to a remote endpoint (using a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Transport}{Transport}}) and sends method invocations to a remote object, using the connection.

The \mbox{\hyperlink{namespacePoco_1_1RemotingNG}{Remoting\+NG}} code generator will generate subclasses of \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Proxy}{Proxy}} creating appropriate serialization and deserialization code for all service object methods.
