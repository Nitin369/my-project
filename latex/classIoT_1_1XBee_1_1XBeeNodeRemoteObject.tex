\doxysubsection{Detailed Description}
This class provides a high-\/level interface to a Digi \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} device using the Digi \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} A\+PI frame-\/based protocol. This class can be used with all \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} devices supporting the \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} A\+PI frame-\/based protocol, including Zig\+Bee devices. Actually supported methods and events are depending on the kind of \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} device, e.\+g., an 802.\+15.\+4 module won\textquotesingle{}t support Zig\+Bee-\/specific A\+PI frames, and vice-\/versa. Please refer to the \mbox{\hyperlink{namespaceIoT_1_1XBee}{X\+Bee}} product manual for detailed information about the A\+PI.
