\doxysubsection{Detailed Description}
This streambuf base32-\/decodes all data read from the istream connected to it.

Note\+: For performance reasons, the characters are read directly from the given istream\textquotesingle{}s underlying streambuf, so the state of the istream will not reflect that of its streambuf.
