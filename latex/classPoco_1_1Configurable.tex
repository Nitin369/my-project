\doxysubsection{Detailed Description}
A simple interface that defines \mbox{\hyperlink{classPoco_1_1Configurable_a565940a99f096c005743a68040ca5e25}{get\+Property()}} and \mbox{\hyperlink{classPoco_1_1Configurable_ae31289819bd01f4a1bda0be67144d5bf}{set\+Property()}} methods.

This interface is implemented by \mbox{\hyperlink{classPoco_1_1Formatter}{Formatter}} and \mbox{\hyperlink{classPoco_1_1Channel}{Channel}} and is used to configure arbitrary channels and formatters.

A property is basically a name-\/value pair. For simplicity, both names and values are strings. Every property controls a certain aspect of a \mbox{\hyperlink{classPoco_1_1Formatter}{Formatter}} or \mbox{\hyperlink{classPoco_1_1Channel}{Channel}}. For example, the \mbox{\hyperlink{classPoco_1_1PatternFormatter}{Pattern\+Formatter}}\textquotesingle{}s formatting pattern is set via a property.

N\+O\+TE\+: The following property names are use internally by the logging framework and must not be used by channels or formatters\+:
\begin{DoxyItemize}
\item class
\item pattern (\mbox{\hyperlink{classPoco_1_1Channel}{Channel}})
\item formatter (\mbox{\hyperlink{classPoco_1_1Channel}{Channel}}) 
\end{DoxyItemize}
