\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Condition}{Condition}} is a synchronization object used to block a thread until a particular condition is met. A \mbox{\hyperlink{classPoco_1_1Condition}{Condition}} object is always used in conjunction with a \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}} (or \mbox{\hyperlink{classPoco_1_1FastMutex}{Fast\+Mutex}}) object.

\mbox{\hyperlink{classPoco_1_1Condition}{Condition}} objects are similar to P\+O\+S\+IX condition variables, which the difference that \mbox{\hyperlink{classPoco_1_1Condition}{Condition}} is not subject to spurious wakeups.

Threads waiting on a \mbox{\hyperlink{classPoco_1_1Condition}{Condition}} are resumed in F\+I\+FO order.
