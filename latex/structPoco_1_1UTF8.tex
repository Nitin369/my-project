\doxysubsection{Detailed Description}
This class provides static methods that are U\+T\+F-\/8 capable variants of the same functions in \mbox{\hyperlink{String_8h}{Poco/\+String.\+h}}.

The various variants of \mbox{\hyperlink{structPoco_1_1UTF8_a0952766010bae80e300b992c07cbb77f}{icompare()}} provide case insensitive comparison for U\+T\+F-\/8 encoded strings.

\mbox{\hyperlink{structPoco_1_1UTF8_a21a7f2f58b199f9ed44c1664ac7f4673}{to\+Upper()}}, \mbox{\hyperlink{structPoco_1_1UTF8_a89268fb9545d1710cbb67de80e19a228}{to\+Upper\+In\+Place()}}, \mbox{\hyperlink{structPoco_1_1UTF8_a6f449cca10275f38a4d6c525ac05ed59}{to\+Lower()}} and \mbox{\hyperlink{structPoco_1_1UTF8_a45446e355d335b8a1bd74e9d9ffb2967}{to\+Lower\+In\+Place()}} provide Unicode-\/based character case transformation for U\+T\+F-\/8 encoded strings.

\mbox{\hyperlink{structPoco_1_1UTF8_a3d8dfc1634a053b1a3f5aeadb3bf0312}{remove\+B\+O\+M()}} removes the U\+T\+F-\/8 Byte Order Mark sequence (0x\+EF, 0x\+BB, 0x\+BF) from the beginning of the given string, if it\textquotesingle{}s there.
