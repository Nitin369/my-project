\doxysubsection{Detailed Description}
This class implements O\+Auth 1.\+0A authentication for H\+T\+TP requests, according to R\+FC 5849.

Only P\+L\+A\+I\+N\+T\+E\+XT and H\+M\+A\+C-\/\+S\+H\+A1 signature methods are supported. The R\+S\+A-\/\+S\+H\+A1 signature method is not supported.

The \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials}{O\+Auth10\+Credentials}} can be used to sign a client request, as well as to verify the signature of a request on the server.

To sign a client request, using a known consumer (client) key, consumer (client) secret, O\+Auth token and token secret\+:


\begin{DoxyEnumerate}
\item Create an \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials}{O\+Auth10\+Credentials}} object using all four credentials, either using the four argument constructor, or by using the default constructor and setting the credentials using the setter methods.
\item Create a \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} containing the full request \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}.
\item Create an appropriate \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} object.
\item Optionally, create a \mbox{\hyperlink{classPoco_1_1Net_1_1HTMLForm}{H\+T\+M\+L\+Form}} object containing additional parameters to sign.
\item Sign the request by calling \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials_a952c94d71c321d95a82e3f90a0151c7a}{authenticate()}}. This will add an O\+Auth Authorization header to the request.
\item Send the request using a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPClientSession}{H\+T\+T\+P\+Client\+Session}}.
\end{DoxyEnumerate}

To request the O\+Auth request token from a server, using only the consumer (client) key and consumer (client) secret\+:


\begin{DoxyEnumerate}
\item Create an \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials}{O\+Auth10\+Credentials}} object using the two consumer credentials, either using the two argument constructor, or by using the default constructor and setting the credentials using the setter methods.
\item Specify the callback \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} using \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials_a96f3ac50b364ed689e4406e2ae019a6d}{set\+Callback()}}.
\item Create a \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} containing the full request \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} to obtain the token.
\item Create an appropriate \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} object.
\item Sign the request by calling \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials_a952c94d71c321d95a82e3f90a0151c7a}{authenticate()}}. This will add an O\+Auth Authorization header to the request.
\item Send the request using a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPClientSession}{H\+T\+T\+P\+Client\+Session}}.
\item The response will contain the request token and request token secret. These can be extracted from the response body using a \mbox{\hyperlink{classPoco_1_1Net_1_1HTMLForm}{H\+T\+M\+L\+Form}} object.
\end{DoxyEnumerate}

Requesting the access token and secret (temporary credentials) from the server is analogous to signing a client request using consumer key, consumer secret, request token and request token secret. The server response will contain the access token and access token secret, which can again be extracted from the response body using a \mbox{\hyperlink{classPoco_1_1Net_1_1HTMLForm}{H\+T\+M\+L\+Form}} object.

To verify a request on the server\+:


\begin{DoxyEnumerate}
\item Create an \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials}{O\+Auth10\+Credentials}} object using the constructor taking a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} object. This will extract the consumer key and token (if provided).
\item Provide the consumer secret and token secret (if required) matching the consumer key and token to the \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials}{O\+Auth10\+Credentials}} object using the setter methods.
\item Create an \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} object containing the full request \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}}.
\item Call \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials_a73d8fcd7842a224fdf4e3ec0f0276cb0}{verify()}} to verify the signature.
\item If verification was successful, and the request was a request for a request (temporary) token, call \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth10Credentials_a493c7b98538ac9060947a208d6e1cf3d}{get\+Callback()}} to obtain the callback \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} provided by the client. 
\end{DoxyEnumerate}
