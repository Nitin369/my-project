\doxysubsection{Detailed Description}
This class implements a \mbox{\hyperlink{classPoco_1_1Net_1_1WebSocket}{Web\+Socket}} according to the \mbox{\hyperlink{classPoco_1_1Net_1_1WebSocket}{Web\+Socket}} protocol specification in R\+FC 6455.

Both client-\/side and server-\/side Web\+Sockets are supported.

Server-\/side Web\+Sockets are usually created from within a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler}{H\+T\+T\+P\+Request\+Handler}}.

Client-\/side Web\+Sockets are created using a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPClientSession}{H\+T\+T\+P\+Client\+Session}}.

Note that special frames like P\+I\+NG must be handled at application level. In the case of a P\+I\+NG, a P\+O\+NG message must be returned.
