\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Util_1_1Timer}{Timer}} allows to schedule tasks (\mbox{\hyperlink{classPoco_1_1Util_1_1TimerTask}{Timer\+Task}} objects) for future execution in a background thread. Tasks may be scheduled for one-\/time execution, or for repeated execution at regular intervals.

The \mbox{\hyperlink{classPoco_1_1Util_1_1Timer}{Timer}} object creates a thread that executes all scheduled tasks sequentially. Therefore, tasks should complete their work as quickly as possible, otherwise subsequent tasks may be delayed.

\mbox{\hyperlink{classPoco_1_1Util_1_1Timer}{Timer}} is save for multithreaded use -\/ multiple threads can schedule new tasks simultaneously.

Acknowledgement\+: The interface of this class has been inspired by the java.\+util.\+Timer class from Java 1.\+3.
