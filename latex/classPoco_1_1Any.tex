\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classPoco_1_1Any}{Any}} class represents a general type and is capable of storing any type, supporting type-\/safe extraction of the internally stored data.

Code taken from the Boost 1.\+33.\+1 library. Original copyright by Kevlin Henney. Modified for \mbox{\hyperlink{namespacePoco}{Poco}} by Applied Informatics.

Modified for small object optimization support (optionally supported through conditional compilation) by Alex Fabijanic.
