\doxysubsection{Detailed Description}
This class is used to split a M\+I\+ME multipart message into its single parts.

The format of multipart messages is described in section 5.\+1 of R\+FC 2046.

To split a multipart message into its parts, do the following\+:
\begin{DoxyItemize}
\item Create a \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartReader}{Multipart\+Reader}} object, passing it an input stream and optionally a boundary string.
\item while \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartReader_ad22900fd1a05c925da24a155d92d6536}{has\+Next\+Part()}} returns true, call \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartReader_aba8888e842e4011b38892312d5bbae7b}{next\+Part()}} and read the part from \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartReader_a478e4b793085b53838089522381c210e}{stream()}}.
\end{DoxyItemize}

Always ensure that you read all data from the part stream, otherwise the \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartReader}{Multipart\+Reader}} will fail to find the next part.
