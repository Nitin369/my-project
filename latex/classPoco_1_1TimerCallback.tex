\subsubsection*{template$<$class C$>$\newline
class Poco\+::\+Timer\+Callback$<$ C $>$}

This template class implements an adapter that sits between a \mbox{\hyperlink{classPoco_1_1Timer}{Timer}} and an object\textquotesingle{}s method invoked by the timer. It is quite similar in concept to the \mbox{\hyperlink{classPoco_1_1RunnableAdapter}{Runnable\+Adapter}}, but provides some \mbox{\hyperlink{classPoco_1_1Timer}{Timer}} specific additional methods. See the \mbox{\hyperlink{classPoco_1_1Timer}{Timer}} class for information on how to use this template class.
