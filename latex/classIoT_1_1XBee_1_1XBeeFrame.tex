Escapes the frame, as required for AP=2 A\+PI mode.

When sending or receiving a U\+A\+RT data frame, specific data values must be escaped (flagged) so they do not interfere with the U\+A\+RT or U\+A\+RT data frame operation. To escape an interfering data byte, insert 0x7D and follow it with the byte to be escaped X\+O\+R’d with 0x20.

Data bytes that need to be escaped\+:
\begin{DoxyItemize}
\item 0x7E – Frame Delimiter
\item 0x7D – Escape
\item 0x11 – X\+ON
\item 0x13 – X\+O\+FF 
\end{DoxyItemize}
