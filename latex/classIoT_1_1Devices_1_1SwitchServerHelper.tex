\doxysubsection{Detailed Description}
The base class for switchable things (that can be turned on and off).

A \mbox{\hyperlink{classIoT_1_1Devices_1_1Switch}{Switch}} has a current state and a target state. Depending upon the implementation, the current state and target state may be different for a short amount of time (e.\+g., while the device is powering on).

In addition to the methods defined in this interface, a \mbox{\hyperlink{classIoT_1_1Devices_1_1Switch}{Switch}} implementation should expose the following properties\+:
\begin{DoxyItemize}
\item display\+State (string, optional)\+: The current state of the \mbox{\hyperlink{classIoT_1_1Devices_1_1Switch}{Switch}}, formatted as string for display purposes. 
\end{DoxyItemize}
