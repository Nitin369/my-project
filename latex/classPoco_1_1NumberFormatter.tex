\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1NumberFormatter}{Number\+Formatter}} class provides static methods for formatting numeric values into strings.

There are two kind of static member functions\+:
\begin{DoxyItemize}
\item format$\ast$ functions return a std\+::string containing the formatted value.
\item append$\ast$ functions append the formatted value to an existing string. 
\end{DoxyItemize}
