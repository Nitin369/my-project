\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Data_1_1RowFilter}{Row\+Filter}} class provides row filtering functionality. A filter contains a set of criteria (field name, value and logical operation) for row filtering. Additionally, a row filter contains a map of pointers to other filters with related logical operations between filters. \mbox{\hyperlink{classPoco_1_1Data_1_1RowFilter}{Row\+Filter}} is typically added to recordset in order to filter its content. Since the recordset own iteration is dependent upon filtering, whenever the filtering criteria is changed, the filter automatically notifies all associated recordsets by rewinding them to the first position.
