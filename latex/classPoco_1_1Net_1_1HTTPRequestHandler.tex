\doxysubsection{Detailed Description}
The abstract base class for H\+T\+T\+P\+Request\+Handlers created by \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServer}{H\+T\+T\+P\+Server}}.

Derived classes must override the \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler_aa766ec81d67103e079501fc102e9fd46}{handle\+Request()}} method. Furthermore, a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandlerFactory}{H\+T\+T\+P\+Request\+Handler\+Factory}} must be provided.

The \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler_aa766ec81d67103e079501fc102e9fd46}{handle\+Request()}} method must perform the complete handling of the H\+T\+TP request connection. As soon as the \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler_aa766ec81d67103e079501fc102e9fd46}{handle\+Request()}} method returns, the request handler object is destroyed.

A new \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequestHandler}{H\+T\+T\+P\+Request\+Handler}} object will be created for each new H\+T\+TP request that is received by the \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServer}{H\+T\+T\+P\+Server}}.
