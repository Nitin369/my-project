\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Data_1_1SessionFactory}{Session\+Factory}} is a singleton class that stores Connectors and allows to create Sessions of the required type\+: \begin{DoxyVerb}Session ses(SessionFactory::instance().create(connector, connectionString));
\end{DoxyVerb}


where the first param presents the type of session one wants to create (e.\+g. for \mbox{\hyperlink{namespacePoco_1_1Data_1_1SQLite}{S\+Q\+Lite}} one would choose \char`\"{}\+S\+Q\+Lite\char`\"{}) and the second param is the connection string that the connector requires to connect to the database.

A concrete example to open an \mbox{\hyperlink{namespacePoco_1_1Data_1_1SQLite}{S\+Q\+Lite}} database stored in the file \char`\"{}dummy.\+db\char`\"{} would be \begin{DoxyVerb}Session ses(SessionFactory::instance().create(SQLite::Connector::KEY, "dummy.db"));
\end{DoxyVerb}


An even simpler way to create a session is to use the two argument constructor of \mbox{\hyperlink{classPoco_1_1Data_1_1Session}{Session}}, which automatically invokes the \mbox{\hyperlink{classPoco_1_1Data_1_1SessionFactory}{Session\+Factory}}\+: \begin{DoxyVerb} Session ses("SQLite", "dummy.db"); 
\end{DoxyVerb}
