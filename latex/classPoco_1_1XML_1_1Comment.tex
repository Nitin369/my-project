\doxysubsection{Detailed Description}
This interface inherits from \mbox{\hyperlink{classPoco_1_1XML_1_1CharacterData}{Character\+Data}} and represents the content of a comment, i.\+e., all the characters between the starting \textquotesingle{}\textquotesingle{}. Note that this is the definition of a comment in \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}}, and, in practice, H\+T\+ML, although some H\+T\+ML tools may implement the full S\+G\+ML comment structure.
