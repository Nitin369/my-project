\doxysubsection{Detailed Description}
This class implements an \mbox{\hyperlink{classPoco_1_1File}{File}} Transfer Protocol (F\+TP, R\+FC 959) client.

Most of the features of the F\+TP protocol, as specified in R\+FC 959, are supported. Not supported are E\+B\+C\+D\+IC and L\+O\+C\+AL data types and format control and structured files.

Also supported are the E\+P\+RT and E\+P\+SV commands from R\+FC 1738 (F\+TP Extensions for I\+Pv6 and N\+AT). The client will first attempt to use the E\+P\+RT and E\+P\+SV commands. If the server does not supports these commands, the client will fall back to P\+O\+RT and P\+A\+SV.
