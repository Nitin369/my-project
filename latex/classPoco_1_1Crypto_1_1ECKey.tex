\doxysubsection{Detailed Description}
This class stores an EC key pair, consisting of private and public key. Storage of the private key is optional.

If a private key is available, the \mbox{\hyperlink{classPoco_1_1Crypto_1_1ECKey}{E\+C\+Key}} can be used for decrypting data (encrypted with the public key) or computing secure digital signatures.
