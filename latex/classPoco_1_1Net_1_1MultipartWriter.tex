\doxysubsection{Detailed Description}
This class is used to write M\+I\+ME multipart messages to an output stream.

The format of multipart messages is described in section 5.\+1 of R\+FC 2046.

To create a multipart message, first create a \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartWriter}{Multipart\+Writer}} object. Then, for each part, call \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartWriter_a163051f043aac77e49e8a1228f2a9a08}{next\+Part()}} and write the content to the output stream. Repeat for all parts. After the last part has been written, call \mbox{\hyperlink{classPoco_1_1Net_1_1MultipartWriter_a43790b24cca533dbbb35710b8d6aac40}{close()}} to finish the multipart message.
