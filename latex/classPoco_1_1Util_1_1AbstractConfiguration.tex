\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration}{Abstract\+Configuration}} is an abstract base class for different kinds of configuration data, such as I\+NI files, property files, \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} configuration files or the Windows Registry.

Configuration property keys have a hierarchical format, consisting of names separated by periods. The exact interpretation of key names is up to the actual subclass implementation of \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration}{Abstract\+Configuration}}. Keys are case sensitive.

All public methods are synchronized, so the class is safe for multithreaded use. \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration}{Abstract\+Configuration}} implements reference counting based garbage collection.

Subclasses must override the \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration_a07e5732151cd6056cfa15809f3a5ac55}{get\+Raw()}}, \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration_a620fb5eedbf878c312d7259c9c251509}{set\+Raw()}} and \mbox{\hyperlink{classPoco_1_1Util_1_1AbstractConfiguration_a64885a63b3990485d518b37697aa1057}{enumerate()}} methods.
