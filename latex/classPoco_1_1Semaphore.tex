\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1Semaphore}{Semaphore}} is a synchronization object with the following characteristics\+: A semaphore has a value that is constrained to be a non-\/negative integer and two atomic operations. The allowable operations are V (here called \mbox{\hyperlink{classPoco_1_1Semaphore_ac3dd7edc6c66f77abf524171ea47c9c7}{set()}}) and P (here called \mbox{\hyperlink{classPoco_1_1Semaphore_a26733c3a620f55953572d60692ba2b35}{wait()}}). A V (\mbox{\hyperlink{classPoco_1_1Semaphore_ac3dd7edc6c66f77abf524171ea47c9c7}{set()}}) operation increases the value of the semaphore by one. A P (\mbox{\hyperlink{classPoco_1_1Semaphore_a26733c3a620f55953572d60692ba2b35}{wait()}}) operation decreases the value of the semaphore by one, provided that can be done without violating the constraint that the value be non-\/negative. A P (\mbox{\hyperlink{classPoco_1_1Semaphore_a26733c3a620f55953572d60692ba2b35}{wait()}}) operation that is initiated when the value of the semaphore is 0 suspends the calling thread. The calling thread may continue when the value becomes positive again.
