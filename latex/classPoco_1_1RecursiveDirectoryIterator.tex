\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T\+Trav\+Str = Children\+First\+Traverse$>$\newline
class Poco\+::\+Recursive\+Directory\+Iterator$<$ T\+Trav\+Str $>$}

The \mbox{\hyperlink{classPoco_1_1RecursiveDirectoryIterator}{Recursive\+Directory\+Iterator}} class is used to enumerate all files in a directory and its subdirectories.

\mbox{\hyperlink{classPoco_1_1RecursiveDirectoryIterator}{Recursive\+Directory\+Iterator}} has some limitations\+:
\begin{DoxyItemize}
\item only forward iteration (++) is supported
\item an iterator copied from another one will always point to the same file as the original iterator, even is the original iterator has been advanced (all copies of an iterator share their state with the original iterator)
\end{DoxyItemize}

The class can follow different traversal strategies\+:
\begin{DoxyItemize}
\item depth-\/first strategy;
\item siblings-\/first strategy. The stategies are set by template parameter. There are two corresponding typedefs\+:
\item Simple\+Recursive\+Directory\+Iterator;
\item Siblings\+First\+Recursive\+Directory\+Iterator.
\end{DoxyItemize}

The depth of traversal can be limited by constructor parameter max\+Depth (which sets the infinite depth by default).
