\subsubsection*{template$<$typename T$>$\newline
struct Poco\+::\+Align\+Of$<$ T $>$}

A templated class that contains an enum value representing the alignment of the template argument. For example, \mbox{\hyperlink{structPoco_1_1AlignOf_a45358cb6efb532d17238b756a12f6fe5ac6623e7d6bd6dbf0dcb112f45e4e00c2}{Align\+Of$<$int$>$\+::\+Alignment}} represents the alignment of type \char`\"{}int\char`\"{}. The alignment calculated is the minimum alignment, and not necessarily the \char`\"{}desired\char`\"{} alignment returned by G\+CC\textquotesingle{}s {\bfseries{alignof}} (for example). Note that because the alignment is an enum value, it can be used as a compile-\/time constant (e.\+g., for template instantiation).
