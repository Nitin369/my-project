\subsubsection*{template$<$class T\+Key, class T\+Value, class T\+Mutex = Fast\+Mutex, class T\+Event\+Mutex = Fast\+Mutex$>$\newline
class Poco\+::\+Access\+Expire\+Cache$<$ T\+Key, T\+Value, T\+Mutex, T\+Event\+Mutex $>$}

An \mbox{\hyperlink{classPoco_1_1AccessExpireCache}{Access\+Expire\+Cache}} caches entries for a fixed time period (per default 10 minutes). Entries expire when they are not accessed with \mbox{\hyperlink{classPoco_1_1AbstractCache_a7f812c1ea00ed55ed9364c28cd32cb33}{get()}} during this time period. Each access resets the start time for expiration. Be careful when using an \mbox{\hyperlink{classPoco_1_1AccessExpireCache}{Access\+Expire\+Cache}}. A cache is often used like cache.\+has(x) followed by cache.\+get x). Note that it could happen that the \char`\"{}has\char`\"{} call works, then the current execution thread gets descheduled, time passes, the entry gets invalid, thus leading to an empty \mbox{\hyperlink{classPoco_1_1SharedPtr}{Shared\+Ptr}} being returned when \char`\"{}get\char`\"{} is invoked.
