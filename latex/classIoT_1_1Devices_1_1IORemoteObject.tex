\doxysubsection{Detailed Description}
The interface for general purpose input/output (G\+P\+IO) ports.

This class represents a single G\+P\+IO pin. Mapping to physical pins is configured when setting up the specific \mbox{\hyperlink{classIoT_1_1Devices_1_1IO}{IO}} implementation class, typically using a configuration file.

Implementations supporting dynamically changing pin directions should expose a string property named \char`\"{}direction\char`\"{} that takes the values \char`\"{}in\char`\"{} and \char`\"{}out\char`\"{}.
