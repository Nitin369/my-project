\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1CharacterData}{Character\+Data}} interface extends \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}} with a set of attributes and methods for accessing character data in the D\+OM. For clarity this set is defined here rather than on each object that uses these attributes and methods. No D\+OM objects correspond directly to \mbox{\hyperlink{classPoco_1_1XML_1_1CharacterData}{Character\+Data}}, though \mbox{\hyperlink{classPoco_1_1XML_1_1Text}{Text}} and others do inherit the interface from it. All offsets in this interface start from 0.

\mbox{\hyperlink{classPoco_1_1XML_1_1Text}{Text}} strings in the D\+OM are represented in either U\+T\+F-\/8 (if X\+M\+L\+\_\+\+U\+N\+I\+C\+O\+D\+E\+\_\+\+W\+C\+H\+A\+R\+\_\+T is not defined) or in U\+T\+F-\/16 (if X\+M\+L\+\_\+\+U\+N\+I\+C\+O\+D\+E\+\_\+\+W\+C\+H\+A\+R\+\_\+T is defined). Indexing on character data is done in X\+M\+L\+Char units.
