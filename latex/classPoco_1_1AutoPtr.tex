\subsubsection*{template$<$class C$>$\newline
class Poco\+::\+Auto\+Ptr$<$ C $>$}

\mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} is a \char`\"{}smart\char`\"{} pointer for classes implementing reference counting based garbage collection. To be usable with the \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} template, a class must implement the following behaviour\+: A class must maintain a reference count. The constructors of the object initialize the reference count to one. The class must implement a public \mbox{\hyperlink{classPoco_1_1AutoPtr_a4551e28d5ee05702d91bc0550dce5186}{duplicate()}} method\+: void \mbox{\hyperlink{classPoco_1_1AutoPtr_a4551e28d5ee05702d91bc0550dce5186}{duplicate()}}; that increments the reference count by one. The class must implement a public release() method\+: void release() that decrements the reference count by one, and, if the reference count reaches zero, deletes the object.

\mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} works in the following way\+: If an \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} is assigned an ordinary pointer to an object (via the constructor or the assignment operator), it takes ownership of the object and the object\textquotesingle{}s reference count remains unchanged. If the \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} is assigned another \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}}, the object\textquotesingle{}s reference count is incremented by one by calling \mbox{\hyperlink{classPoco_1_1AutoPtr_a4551e28d5ee05702d91bc0550dce5186}{duplicate()}} on its object. The destructor of \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} calls release() on its object. \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} supports dereferencing with both the -\/$>$ and the $\ast$ operator. An attempt to dereference a null \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} results in a Null\+Pointer\+Exception being thrown. \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} also implements all relational operators. Note that \mbox{\hyperlink{classPoco_1_1AutoPtr}{Auto\+Ptr}} allows casting of its encapsulated data types.
