\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1Element}{Element}} interface represents an element in an \mbox{\hyperlink{namespacePoco_1_1XML}{X\+ML}} document. Elements may have attributes associated with them; since the \mbox{\hyperlink{classPoco_1_1XML_1_1Element}{Element}} interface inherits from \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}}, the generic \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}} interface attribute attributes may be used to retrieve the set of all attributes for an element. There are methods on the \mbox{\hyperlink{classPoco_1_1XML_1_1Element}{Element}} interface to retrieve either an \mbox{\hyperlink{classPoco_1_1XML_1_1Attr}{Attr}} object by name or an attribute value by name.
