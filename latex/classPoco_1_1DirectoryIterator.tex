\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1DirectoryIterator}{Directory\+Iterator}} class is used to enumerate all files in a directory.

\mbox{\hyperlink{classPoco_1_1DirectoryIterator}{Directory\+Iterator}} has some limitations\+:
\begin{DoxyItemize}
\item only forward iteration (++) is supported
\item an iterator copied from another one will always point to the same file as the original iterator, even is the original iterator has been advanced (all copies of an iterator share their state with the original iterator)
\item because of this you should only use the prefix increment operator 
\end{DoxyItemize}
