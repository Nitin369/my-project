\doxysubsection{Detailed Description}
This class implements an Simple Mail Transfer Protocol (S\+M\+TP, R\+FC 2821) client for sending e-\/mail messages that supports the S\+T\+A\+R\+T\+T\+LS command for secure connections.

Usage is as follows\+:
\begin{DoxyEnumerate}
\item Create a \mbox{\hyperlink{classPoco_1_1Net_1_1SecureSMTPClientSession}{Secure\+S\+M\+T\+P\+Client\+Session}} object.
\item Call \mbox{\hyperlink{classPoco_1_1Net_1_1SMTPClientSession_aab851514102f94509d3b4e371c103b1b}{login()}} or login(hostname).
\item Call \mbox{\hyperlink{classPoco_1_1Net_1_1SecureSMTPClientSession_a3efbaec80f8cf26401a2bb4ce2fbb706}{start\+T\+L\+S()}} to switch to a secure connection. Check the return value to see if a secure connection has actually been established (not all servers may support S\+T\+A\+R\+T\+T\+LS).
\item Call any of the \mbox{\hyperlink{classPoco_1_1Net_1_1SMTPClientSession_aab851514102f94509d3b4e371c103b1b}{login()}} methods to securely authenticate with a username and password.
\item Send the message(s). 
\end{DoxyEnumerate}
