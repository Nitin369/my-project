\subsubsection*{template$<$typename ch, typename tr$>$\newline
class Poco\+::\+Basic\+Unbuffered\+Stream\+Buf$<$ ch, tr $>$}

This is an implementation of an unbuffered streambuf that greatly simplifies the implementation of custom streambufs of various kinds. Derived classes only have to override the methods read\+From\+Device() or write\+To\+Device().
