class Poco\+::\+Remoting\+N\+G\+::\+Type\+Deserializer$<$ T $>$}

\mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeDeserializer}{Type\+Deserializer}} is used to deserialize the content of a variable for transportation over the network.

The default \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeDeserializer}{Type\+Deserializer}} template simply forwards \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeDeserializer_a93de3b10c219daf40047332e26ac0442}{deserialize()}} calls to a \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Deserializer}{Deserializer}}. This means that deserializaation using the default template only works for types directly supported by \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Deserializer}{Deserializer}}.

Serialization of complex types can be done by providing specializations of the \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1TypeDeserializer}{Type\+Deserializer}} class template.
