\doxysubsection{Detailed Description}
This class implements a streambuf interface to a \mbox{\hyperlink{classPoco_1_1Logger}{Logger}}.

The streambuf appends all characters written to it to a string. As soon as a CR or LF (std\+::endl) is written, the string is sent to the \mbox{\hyperlink{classPoco_1_1Logger}{Logger}}, with the set priority.
