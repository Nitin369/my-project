\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1Util_1_1LoggingSubsystem}{Logging\+Subsystem}} class initializes the logging framework using the \mbox{\hyperlink{classPoco_1_1Util_1_1LoggingConfigurator}{Logging\+Configurator}}.

It also sets the \mbox{\hyperlink{classPoco_1_1Util_1_1Application}{Application}}\textquotesingle{}s logger to the logger specified by the \char`\"{}application.\+logger\char`\"{} property, or to \char`\"{}\+Application\char`\"{} if the property is not specified.
