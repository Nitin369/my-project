\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1NumberParser}{Number\+Parser}} class provides static methods for parsing numbers out of strings.

Note that leading or trailing whitespace is not allowed in the string. \mbox{\hyperlink{namespacePoco_a43e5bb0afe99411c32a846b125595106}{Poco\+::trim()}} or \mbox{\hyperlink{namespacePoco_ac4e17d0ed49fbb599d7159d83db33ca7}{Poco\+::trim\+In\+Place()}} can be used to remove leading or trailing whitespace.
