\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1FastMutex}{Fast\+Mutex}} (mutual exclusion) is similar to a \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}}. Unlike a \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}}, however, a \mbox{\hyperlink{classPoco_1_1FastMutex}{Fast\+Mutex}} is not recursive, which means that a deadlock will occur if the same thread tries to lock a mutex it has already locked again. Locking a \mbox{\hyperlink{classPoco_1_1FastMutex}{Fast\+Mutex}} is faster than locking a recursive \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}}. Using the \mbox{\hyperlink{classPoco_1_1ScopedLock}{Scoped\+Lock}} class is the preferred way to automatically lock and unlock a mutex.
