\doxysubsection{Detailed Description}
This stream decompresses all data passing through it using zlib\textquotesingle{}s inflate algorithm.

After all data has been written to the stream, \mbox{\hyperlink{classPoco_1_1InflatingOutputStream_ae4305306ac44110eb9a7ff06813398d4}{close()}} must be called to ensure completion of decompression.
