\doxysubsection{Detailed Description}
An unidirectional iterator for iterating over characters in a string. The \mbox{\hyperlink{classPoco_1_1TextIterator}{Text\+Iterator}} uses a \mbox{\hyperlink{classPoco_1_1TextEncoding}{Text\+Encoding}} object to work with multi-\/byte character encodings like U\+T\+F-\/8. Characters are reported in \mbox{\hyperlink{classPoco_1_1Unicode}{Unicode}}.

Example\+: Count the number of U\+T\+F-\/8 characters in a string. \begin{DoxyVerb}UTF8Encoding utf8Encoding;
std::string utf8String("....");
TextIterator it(utf8String, utf8Encoding);
TextIterator end(utf8String);
int n = 0;
while (it != end) { ++n; ++it; }
\end{DoxyVerb}


N\+O\+TE\+: When an U\+T\+F-\/16 encoding is used, surrogate pairs will be reported as two separate characters, due to restrictions of the \mbox{\hyperlink{classPoco_1_1TextEncoding}{Text\+Encoding}} class.

For iterating over char buffers, see the \mbox{\hyperlink{classPoco_1_1TextBufferIterator}{Text\+Buffer\+Iterator}} class.
