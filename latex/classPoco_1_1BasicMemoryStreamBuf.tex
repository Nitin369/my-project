\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename ch, typename tr$>$\newline
class Poco\+::\+Basic\+Memory\+Stream\+Buf$<$ ch, tr $>$}

\mbox{\hyperlink{classPoco_1_1BasicMemoryStreamBuf}{Basic\+Memory\+Stream\+Buf}} is a simple implementation of a stream buffer for reading and writing from a memory area.

This streambuf only supports unidirectional streams. In other words, the \mbox{\hyperlink{classPoco_1_1BasicMemoryStreamBuf}{Basic\+Memory\+Stream\+Buf}} can be used for the implementation of an istream or an ostream, but not for an iostream.
