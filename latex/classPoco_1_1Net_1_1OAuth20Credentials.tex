\doxysubsection{Detailed Description}
This class implements O\+Auth 2.\+0 authentication for H\+T\+TP requests, via Bearer tokens in the Authorization header, according to R\+FC 6749 and R\+FC 6750.

To add an Authorization header containing a bearer token to a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} object, create an \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth20Credentials}{O\+Auth20\+Credentials}} object with the bearer token and call \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth20Credentials_a4747627648320f5eba59c6ca64747c3d}{authenticate()}}.

The bearer token can also be extracted from a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} by creating the \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth20Credentials}{O\+Auth20\+Credentials}} object with a \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPRequest}{H\+T\+T\+P\+Request}} object containing a \char`\"{}\+Bearer\char`\"{} Authorization header and calling \mbox{\hyperlink{classPoco_1_1Net_1_1OAuth20Credentials_ae3b657deea475ecbb6306c58dbdb16f7}{get\+Bearer\+Token()}}.

The authorization header scheme can be changed from \char`\"{}\+Bearer\char`\"{} to a custom value. For example, Git\+Hub uses the \char`\"{}token\char`\"{} scheme.
