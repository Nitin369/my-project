\doxysubsection{Detailed Description}
The base class for all part or attachment handlers. \begin{DoxyVerb}Part handlers are used for handling email parts and 
attachments in MIME multipart messages, as well as file 
uploads via HTML forms.

Subclasses must override handlePart(). 
\end{DoxyVerb}
