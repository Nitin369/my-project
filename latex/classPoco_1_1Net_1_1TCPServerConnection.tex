\doxysubsection{Detailed Description}
The abstract base class for T\+CP server connections created by \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServer}{T\+C\+P\+Server}}.

Derived classes must override the \mbox{\hyperlink{classPoco_1_1Runnable_ad89816e81207bf5f1e52d5df860d11d4}{run()}} method (inherited from \mbox{\hyperlink{classPoco_1_1Runnable}{Runnable}}). Furthermore, a \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServerConnectionFactory}{T\+C\+P\+Server\+Connection\+Factory}} must be provided for the subclass.

The \mbox{\hyperlink{classPoco_1_1Runnable_ad89816e81207bf5f1e52d5df860d11d4}{run()}} method must perform the complete handling of the client connection. As soon as the \mbox{\hyperlink{classPoco_1_1Runnable_ad89816e81207bf5f1e52d5df860d11d4}{run()}} method returns, the server connection object is destroyed and the connection is automatically closed.

A new \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServerConnection}{T\+C\+P\+Server\+Connection}} object will be created for each new client connection that is accepted by \mbox{\hyperlink{classPoco_1_1Net_1_1TCPServer}{T\+C\+P\+Server}}.
