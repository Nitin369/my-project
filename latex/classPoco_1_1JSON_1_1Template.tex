\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1JSON_1_1Template}{Template}} is a template engine which uses \mbox{\hyperlink{namespacePoco_1_1JSON}{J\+S\+ON}} as input for generating output. There are commands for looping over \mbox{\hyperlink{namespacePoco_1_1JSON}{J\+S\+ON}} arrays, include other templates, conditional output, etc.

All text is send to the outputstream. A command is placed between $<$? and \hypertarget{classPoco_1_1JSON_1_1Template_autotoc_md2}{}\doxysubsubsection{?$>$}\label{classPoco_1_1JSON_1_1Template_autotoc_md2}
These are the available commands\+: \begin{DoxyVerb}<? echo query ?>
\end{DoxyVerb}


\DoxyHorRuler{0}
 The result of the query is send to the output stream This command can also be written as $<$?= query ?$>$ \begin{DoxyVerb}<? if query ?> <? else ?> <? endif ?>
\end{DoxyVerb}


\DoxyHorRuler{0}
 When the result of query is true, all the text between if and else (or endif when there is no else) is send to the output stream. When the result of query is false, all the text between else and endif is send to the output stream. An empty object, an empty array or a null value is considered as a false value. For numbers a zero is false. An empty String is also false. \begin{DoxyVerb}<? ifexist query ?> <? else ?> <? endif ?>
\end{DoxyVerb}


\DoxyHorRuler{0}
 This can be used to check the existence of the value. Use this for example when a zero value is ok (which returns false for $<$? if ?$>$. \begin{DoxyVerb}<? for variable query ?> <? endfor ?>
\end{DoxyVerb}


\DoxyHorRuler{0}
 The result of the query must be an array. For each element in the array the text between for and endfor is send to the output stream. The active element is stored in the variable. \begin{DoxyVerb}<? include "filename" ?>
\end{DoxyVerb}


\DoxyHorRuler{0}
 Includes a template. When the filename is relative it will try to resolve the filename against the active template. When this file doesn\textquotesingle{}t exist, it can still be found when the J\+S\+O\+N\+Template\+Cache is used.

A query is passed to \mbox{\hyperlink{classPoco_1_1JSON_1_1Query}{Poco\+::\+J\+S\+O\+N\+::\+Query}} to get the value.
