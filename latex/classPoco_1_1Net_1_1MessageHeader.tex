\doxysubsection{Detailed Description}
A collection of name-\/value pairs that are used in various internet protocols like H\+T\+TP and S\+M\+TP.

The name is case-\/insensitive.

There can be more than one name-\/value pair with the same name.

\mbox{\hyperlink{classPoco_1_1Net_1_1MessageHeader}{Message\+Header}} supports writing and reading the header data in R\+FC 2822 format.

The maximum number of fields can be restricted by calling \mbox{\hyperlink{classPoco_1_1Net_1_1MessageHeader_a6bc78e56e2faa12781407fe2ef1f39f8}{set\+Field\+Limit()}}. This is useful to defend against certain kinds of denial-\/of-\/service attacks. The limit is only enforced when parsing header fields from a stream, not when programmatically adding them. The default limit is 100.
