\subsubsection*{template$<$class B$>$\newline
class Poco\+::\+Abstract\+Meta\+Object$<$ B $>$}

A \mbox{\hyperlink{classPoco_1_1MetaObject}{Meta\+Object}} stores some information about a C++ class. The \mbox{\hyperlink{classPoco_1_1MetaObject}{Meta\+Object}} class is used by the \mbox{\hyperlink{classPoco_1_1Manifest}{Manifest}} class. \mbox{\hyperlink{classPoco_1_1AbstractMetaObject}{Abstract\+Meta\+Object}} is a common base class for all \mbox{\hyperlink{classPoco_1_1MetaObject}{Meta\+Object}} in a rooted class hierarchy. A \mbox{\hyperlink{classPoco_1_1MetaObject}{Meta\+Object}} can also be used as an object factory for its class.
