\subsubsection*{template$<$typename T$>$\newline
class Poco\+::\+Dynamic\+::\+Var\+Holder\+Impl$<$ T $>$}

Template based implementation of a \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolder}{Var\+Holder}}. This class provides type storage for user-\/defined types that do not have \mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolderImpl}{Var\+Holder\+Impl}} specialization.

The actual conversion work happens in the template specializations of this class.

\mbox{\hyperlink{classPoco_1_1Dynamic_1_1VarHolderImpl}{Var\+Holder\+Impl}} throws following exceptions\+: Bad\+Cast\+Exception (if the requested conversion is not implemented) Range\+Exception (if an attempt is made to assign a numeric value outside of the target min/max limits Syntax\+Exception (if an attempt is made to convert a string containing non-\/numeric characters to number)

In order to support efficient direct extraction of the held value, all specializations must additionally implement a public member function\+: \begin{DoxyVerb}const T& value() const
\end{DoxyVerb}


returning a const reference to the actual stored value.
