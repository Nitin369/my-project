\doxysubsection{Detailed Description}
A Stream\+Converter converts streams from one encoding (in\+Encoding) into another (out\+Encoding). If a character cannot be represented in out\+Encoding, default\+Char is used instead. If a byte sequence is not valid in in\+Encoding, default\+Char is used instead and the encoding error count is incremented.
