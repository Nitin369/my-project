The base class for triggers, such as push buttons or motion detectors.

In addition to the methods defined in this interface, a \mbox{\hyperlink{classIoT_1_1Devices_1_1Trigger}{Trigger}} implementation should expose the following properties\+:
\begin{DoxyItemize}
