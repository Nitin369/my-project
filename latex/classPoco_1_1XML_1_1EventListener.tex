\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1EventListener}{Event\+Listener}} interface is the primary method for handling events. Users implement the \mbox{\hyperlink{classPoco_1_1XML_1_1EventListener}{Event\+Listener}} interface and register their listener on an \mbox{\hyperlink{classPoco_1_1XML_1_1EventTarget}{Event\+Target}} using the Add\+Event\+Listener method. The users should also remove their \mbox{\hyperlink{classPoco_1_1XML_1_1EventListener}{Event\+Listener}} from its \mbox{\hyperlink{classPoco_1_1XML_1_1EventTarget}{Event\+Target}} after they have completed using the listener.

When a \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}} is copied using the clone\+Node method the Event\+Listeners attached to the source \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}} are not attached to the copied \mbox{\hyperlink{classPoco_1_1XML_1_1Node}{Node}}. If the user wishes the same Event\+Listeners to be added to the newly created copy the user must add them manually.
