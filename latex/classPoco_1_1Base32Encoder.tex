\doxysubsection{Detailed Description}
This ostream base32-\/encodes all data written to it and forwards it to a connected ostream. Always call \mbox{\hyperlink{classPoco_1_1Base32EncoderIOS_a5c3d279090d3832222ff88e68b8b2ae3}{close()}} when done writing data, to ensure proper completion of the encoding operation.

Note\+: The characters are directly written to the ostream\textquotesingle{}s streambuf, thus bypassing the ostream. The ostream\textquotesingle{}s state is therefore not updated to match the buffer\textquotesingle{}s state.
