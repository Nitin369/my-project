\doxysubsection{Detailed Description}
Objects implementing the \mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}} interface are used to represent collections of nodes that can be accessed by name. Note that \mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}} does not inherit from \mbox{\hyperlink{classPoco_1_1XML_1_1NodeList}{Node\+List}}; Named\+Node\+Maps are not maintained in any particular order. Objects contained in an object implementing \mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}} may also be accessed by an ordinal index, but this is simply to allow convenient enumeration of the contents of a \mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}}, and does not imply that the D\+OM specifies an order to these Nodes.

\mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}} objects in the D\+OM are live.

A \mbox{\hyperlink{classPoco_1_1XML_1_1NamedNodeMap}{Named\+Node\+Map}} returned from a method must be released with a call to \mbox{\hyperlink{classPoco_1_1XML_1_1DOMObject_a59fcc225a5bcf0fcf7e49d06ecba249a}{release()}} when no longer needed.
