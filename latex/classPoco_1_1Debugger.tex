\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1Debugger}{Debugger}} class provides an interface to the debugger. The presence of a debugger can be checked for, messages can be written to the debugger\textquotesingle{}s log window and a break into the debugger can be enforced. The methods only work if the program is compiled in debug mode (the macro \+\_\+\+D\+E\+B\+UG is defined).
