\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1XML_1_1DocumentEvent}{Document\+Event}} interface provides a mechanism by which the user can create an \mbox{\hyperlink{classPoco_1_1XML_1_1Event}{Event}} of a type supported by the implementation. It is expected that the \mbox{\hyperlink{classPoco_1_1XML_1_1DocumentEvent}{Document\+Event}} interface will be implemented on the same object which implements the \mbox{\hyperlink{classPoco_1_1XML_1_1Document}{Document}} interface in an implementation which supports the \mbox{\hyperlink{classPoco_1_1XML_1_1Event}{Event}} model.
