\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1OSP_1_1BundleStorage}{Bundle\+Storage}} manages provides access to all resources stored inside of a bundle.

\mbox{\hyperlink{classPoco_1_1OSP_1_1BundleStorage}{Bundle\+Storage}} itself is an abstract class; subclasses implement the \mbox{\hyperlink{classPoco_1_1OSP_1_1BundleStorage}{Bundle\+Storage}} interface for bundles stored in directories or \mbox{\hyperlink{namespacePoco_1_1Zip}{Zip}} files.

Paths to resources within a bundle are always specified using Unix notation -\/ directories are separated with a forward slash (\textquotesingle{}/\textquotesingle{}).
