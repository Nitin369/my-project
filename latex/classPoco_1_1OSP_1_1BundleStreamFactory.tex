\doxysubsection{Detailed Description}
An implementation of the \mbox{\hyperlink{classPoco_1_1URIStreamFactory}{U\+R\+I\+Stream\+Factory}} interface that handles \mbox{\hyperlink{classPoco_1_1OSP_1_1Bundle}{Bundle}} (bndl) U\+R\+Is for accessing resources stored in bundles.

The format of a bndl \mbox{\hyperlink{classPoco_1_1URI}{U\+RI}} is\+: bndl\+://$<$symbolic-\/name$>$/$<$resource-\/path$>$

Example\+: bndl\+://com.appinf.\+osp.\+sample-\/bundle/sample.\+txt
