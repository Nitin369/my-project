\doxysubsection{Detailed Description}
This class represents a bundle version number in \mbox{\hyperlink{namespacePoco_1_1OSP}{O\+SP}}. A version number consists of a major version number, a minor version number and a revision number. Optionally, a release designation can be specified.

A \mbox{\hyperlink{classPoco_1_1OSP_1_1Version}{Version}} object can be formatted as a string. Also, a \mbox{\hyperlink{classPoco_1_1OSP_1_1Version}{Version}} object can be constructed by parsing a string.

The string representation of a version number is $<$major$>$.$<$minor$>$.$<$revision$>$\mbox{[}-\/$<$designation$>$\mbox{]}

In a comparison, only major, minor and revision number are considered. The release designation is ignored.
