\doxysubsection{Detailed Description}
This adapter simplifies using static member functions as well as standalone functions as targets for threads. Note that it is possible to pass those entities directly to \mbox{\hyperlink{classPoco_1_1Thread_aca167199ed123ca70ef94b166806321a}{Thread\+::start()}}. This adapter is provided as a convenience for higher abstraction level scenarios where \mbox{\hyperlink{classPoco_1_1Runnable}{Runnable}} abstract class is used.

For using a non-\/static member function as a thread target, please see the \mbox{\hyperlink{classPoco_1_1RunnableAdapter}{Runnable\+Adapter}} class.
