\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1Net_1_1HTTPSession}{H\+T\+T\+P\+Session}} implements basic H\+T\+TP session management for both H\+T\+TP clients and H\+T\+TP servers.

\mbox{\hyperlink{classPoco_1_1Net_1_1HTTPSession}{H\+T\+T\+P\+Session}} implements buffering for H\+T\+TP connections, as well as specific support for the various H\+T\+TP stream classes.

This class can not be instantiated. \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPClientSession}{H\+T\+T\+P\+Client\+Session}} or \mbox{\hyperlink{classPoco_1_1Net_1_1HTTPServerSession}{H\+T\+T\+P\+Server\+Session}} must be used instead.
