\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Context}{Context}} object is used to pass transport-\/specific information (such as caller IP address or authentication information) to a service method in a generic way.

\mbox{\hyperlink{classPoco_1_1RemotingNG_1_1Context}{Context}} objects are created and filled by the transport, and are accessed via thread-\/local storage.
