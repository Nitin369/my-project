\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPoco_1_1ArchiveStrategy}{Archive\+Strategy}} is used by \mbox{\hyperlink{classPoco_1_1FileChannel}{File\+Channel}} to rename a rotated log file for archiving.

Archived files can be automatically compressed, using the gzip file format.
