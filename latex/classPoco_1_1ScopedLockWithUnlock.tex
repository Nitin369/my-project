\doxysubsection{Detailed Description}
\subsubsection*{template$<$class M$>$\newline
class Poco\+::\+Scoped\+Lock\+With\+Unlock$<$ M $>$}

A class that simplifies thread synchronization with a mutex. The constructor accepts a \mbox{\hyperlink{classPoco_1_1Mutex}{Mutex}} (and optionally a timeout value in milliseconds) and locks it. The destructor unlocks the mutex. The \mbox{\hyperlink{classPoco_1_1ScopedLockWithUnlock_aed2428d93a24025eaf802a938aa60d45}{unlock()}} member function allows for manual unlocking of the mutex.
