\doxysubsection{Detailed Description}
\mbox{\hyperlink{classPoco_1_1OSP_1_1CodeCache}{Code\+Cache}} is a utility class that manages the code cache directory where the shared libraries of resolved bundles are kept.

Under certain circumstances, multiple \mbox{\hyperlink{namespacePoco_1_1OSP}{O\+SP}} processes can shared the same code cache directory. This is not recommended, but may be done to optimize a system\textquotesingle{}s resource usage. If the code\+Cache is shared between multiple processes, all processes must use the same versions of common bundles. In other words, while different processes can run different sets of bundles, all bundles used by multiple processes must be the same, in order to prevent conflicts caused by incompatible shared libraries in the common code cache.

See the \mbox{\hyperlink{classPoco_1_1OSP_1_1CodeCache}{Code\+Cache}} constructor for enabling support for a shared code cache.
